# 中华全国律师协会律师担任破产管理人业务操作指引

> **发文机关**：中华全国律师协会
> **时效性**：现行有效
> **章节结构**：六章（一般规定、裁定受理破产清算申请后的管理人职责、裁定重整后的管理人职责、裁定和解后的管理人职责、宣告破产后的管理人职责、管理人终止执行职务）

## 文书要点

《中华全国律师协会律师担任破产管理人业务操作指引》是全国律协为指导律师担任破产管理人业务而制定的行业自律规范。本指引为律师担任破产管理人业务的指导性意见，与相关法律、行政法规及司法解释有抵触的，以相关法律、行政法规及司法解释的规定为准。

本指引适用于进入人民法院公布的机构管理人名册的律师事务所，以及进入人民法院公布的个人管理人名册的律师个人，接受人民法院指定担任破产案件管理人并依法履行管理人职责的情形。

---

## 全文

### 第一章 一般规定

**第1条 宗旨和效力**

1.1 为指导律师担任破产管理人业务，规范律师执业行为，提高律师的服务质量和水平，防范执业风险，充分发挥律师在企业破产案件事务中的作用，依据《中华人民共和国企业破产法》及其他相关法律、行政法规和司法解释的规定，制定本指引。

1.2 本指引为律师担任破产管理人业务的指导性意见。本指引与相关法律、行政法规及司法解释有抵触的，以相关法律、行政法规及司法解释的规定为准。

**第2条 适用范围**

2.1 进入人民法院公布的机构管理人名册的律师事务所，以及进入人民法院公布的个人管理人名册的律师个人，接受人民法院指定担任破产案件管理人并依法履行管理人职责的，适用本指引。

2.2 人民法院指定清算组为管理人的，作为清算组成员的律师事务所或律师个人依法履行管理人职责的，适用本指引。

2.3 被人民法院指定为管理人的律师事务所或者律师个人指派的管理人团队成员，无论其是否为律师，在其履行管理人职责时，负责指派其为成员的律师事务所或者律师个人可以要求其参照本指引执行。

**第3条 相关制度制订**

3.1 进入人民法院公布的机构管理人名册的律师事务所，应当结合本律师事务所的实际情况，制定本律师事务所担任管理人的相关制度。相关制度包括但不限于：管理人团队组成及分工负责制度、管理人业务培训制度、管理人消极资格审查和报告制度、管理人业务操作流程制度、管理人工作底稿和档案管理制度、管理人报酬分配与风险承担制度等。

3.2 进入人民法院公布的个人管理人名册中的律师个人，除应当配合本律师事务所制定前款规定的相关制度外，还应当配合本律师事务所制定律师个人担任管理人业务的相关制度。

3.3 进入人民法院公布的个人管理人名册中的律师个人，应当参加律师执业责任保险。

**第4条 管理人的指定、回避、辞职和更换**

4.1 进入人民法院公布的管理人名册中的律师事务所、律师个人，应当接受人民法院关于破产案件管理人的指定。但被指定的律师事务所、律师个人有下列情形之一的，视为不宜担任管理人，其应当向人民法院申请不接受指定并说明情况：

4.1.1 因故意犯罪受过刑事处罚；

4.1.2 曾被吊销相关专业执业证书；

4.1.3 与破产案件有利害关系；

4.1.4 人民法院认为的不宜担任管理人的其他情形。

4.2 被人民法院指定为破产案件管理人的律师事务所，有下列情形之一，可能影响其忠实履行管理人职责的，视为与破产案件有利害关系：

4.2.1 与债务人、债权人有未了结的债权债务关系；

4.2.2 在人民法院受理破产申请前3年内，曾为债务人提供相对固定的法律服务；

4.2.3 现在是或者在人民法院受理破产申请前3年内曾经是债务人、债权人的控股股东或者实际控制人；

4.2.4 现在担任或者在人民法院受理破产申请前3年内曾经担任债务人、债权人的财务顾问、法律顾问；

4.2.5 人民法院认为可能影响其忠实履行管理人职责的其他情形。

4.3 被人民法院指定为破产案件管理人的律师个人，有下列情形之一，可能影响其忠实履行管理人职责的，视为与破产案件有利害关系：

4.3.1 具有本指引第4.2条款规定的情形；

4.3.2 现在担任或者在人民法院受理破产申请前3年内曾经担任债务人、债权人的董事、监事、高级管理人员；

4.3.3 与债权人或者债务人的控股股东、董事、监事、高级管理人员存在夫妻、直系血亲、三代以内旁系血亲或者近姻亲关系；

4.3.4 人民法院认为可能影响其公平履行管理人职责的其他情形。

4.4 被人民法院指定为破产案件管理人的律师事务所，有下列情形之一的，视为有人民法院认为不宜担任管理人的其他情形：

4.4.1 因执业、经营中故意或者重大过失行为，受到行政机关、监管机构或者行业自律组织行政处罚或者纪律处分之日起未逾3年；

4.4.2 因涉嫌违法行为正被相关部门调查；

4.4.3 执业许可证或者营业执照被吊销或者注销；

4.4.4 因不适当履行职务或者拒绝接受人民法院指定等原因，被人民法院从管理人名册除名之日起未逾3年；

4.4.5 缺乏担任管理人所应具备的专业能力；

4.4.6 缺乏承担民事责任的能力；

4.4.7 有重大债务纠纷；

4.4.8 出现解散、破产事由或者丧失承担执业责任风险的能力；

4.4.9 可能影响履行管理人职责的其他情形。

4.5 被人民法院指定为破产案件管理人的律师个人，有下列情形之一的，视为有人民法院认为不宜担任管理人的其他情形：

4.5.1 因执业中故意或者重大过失行为，受到行政机关、监管机构或者行业自律组织行政处罚或者纪律处分之日起未逾3年；

4.5.2 因涉嫌违法行为正被相关部门调查；

4.5.3 执业证书被吊销或者注销；

4.5.4 因不适当履行职务或者拒绝接受人民法院指定等原因，被人民法院从管理人名册除名之日起未逾3年；

4.5.5 缺乏担任管理人所应具备的专业能力；

4.5.6 缺乏承担民事责任的能力；

4.5.7 有重大债务纠纷；

4.5.8 可能影响履行管理人职责的其他情形。

4.6 进入人民法院公布的管理人名册中的律师事务所、律师个人，在参与人民法院采取公告方式邀请的管理人业务竞争时，如发现自己有本指引第4.1条、第4.2条、第4.3条、第4.4条、第4.5条规定情形之一的，应当主动退出竞争。

4.7 律师事务所、律师个人接受人民法院指定为破产案件管理人后，发现自己有本指引第4.1条、第4.2条、第4.3条、第4.4条、第4.5条规定情形之一的，应当主动申请回避并向人民法院书面说明情况。

4.8 律师事务所、律师个人接受人民法院指定为破产案件管理人后，发现自己与破产案件有利害关系，有本指引第4.2条、第4.3条规定情形之一的，应当向人民法院提出回避申请并说明情况。

4.9 律师事务所、律师个人接受人民法院指定为破产案件管理人后，发现自己有不宜担任管理人的其他情形，有本指引第4.4条、第4.5条规定情形之一的，应当向人民法院提出辞职并说明情况。

4.10 律师事务所、律师个人接受人民法院指定为破产案件管理人后，债权人会议向人民法院申请更换管理人的，如债权人会议申请更换管理人的理由成立，且律师事务所、律师个人确有本指引第4.2条、第4.3条、第4.4条、第4.5条规定情形之一的，律师事务所、律师个人应当主动向人民法院申请辞去管理人职务。

4.11 人民法院发现指定的管理人应当回避、辞职或者被更换的，可以直接作出决定。

4.12 律师事务所、律师个人辞去管理人职务或者被更换的，应当及时向人民法院移交其已接管的全部资料、财产、印章和账簿、文书等，并协助新的管理人接管破产事务。

**第5条 管理人团队**

5.1 接受人民法院指定的律师事务所，应当组建管理人团队，具体负责执行管理人事务。管理人团队应当由律师组成，根据需要可以聘请会计师、清算师、税务师、资产评估师、投资顾问、财务顾问等必要的工作人员。

5.2 接受人民法院指定的律师个人，应当组建管理人团队，具体负责执行管理人事务。管理人团队应当由律师组成，根据需要可以聘请会计师、清算师、税务师、资产评估师、投资顾问、财务顾问等必要的工作人员。

5.3 管理人团队负责人由接受指定的律师事务所或者律师个人确定。

5.4 管理人团队负责人履行以下职责：

5.4.1 根据债务人企业的实际情况，确定管理人团队内部的分工，掌握工作进度，随时处理出现的问题，协调程序中各环节的关系；

5.4.2 确定管理人工作计划，工作计划应包括管理人内部分工，各项工作完成的时间、思路、步骤和程序等；计划要做到全面、完整、分清轻重缓急、切实可行；

5.4.3 负责与人民法院联络，并及时向人民法院报告工作；

5.4.4 代表管理人对外签订合同、参加谈判、出席会议、授权相关人员代表管理人参加诉讼或者仲裁活动；

5.4.5 以管理人名义向人民法院及债权人会议提交各类报告，出席债权人会议并代表管理人做工作报告；

5.4.6 掌管管理人印章；

5.4.7 审批破产费用支出；

5.4.8 召集和主持管理人工作会议，并形成工作会议纪要；

5.4.9 人民法院或者债权人会议认为应当由管理人团队负责人履行的其他职责。

5.5 管理人团队负责人应当将管理人团队组成情况、分工情况及负责人姓名、联系方式报告人民法院。

5.6 管理人团队负责人应当制定管理人团队会议制度，定期召开管理人团队会议，研究解决管理人事务执行中的重大问题。

5.7 管理人团队负责人应当制定管理人团队内部管理制度，明确管理人团队成员的职责分工、工作流程、工作标准、工作纪律等。

5.8 管理人团队负责人应当制定管理人团队工作底稿制度，明确工作底稿的制作、保管、查阅、移交等要求。

5.9 管理人团队负责人应当制定管理人团队档案管理制度，明确档案的收集、整理、保管、移交等要求。

5.10 管理人团队负责人应当制定管理人团队保密制度，明确保密范围、保密措施、保密责任等。

5.11 管理人团队负责人应当制定管理人团队廉洁制度，明确廉洁要求、廉洁措施、廉洁责任等。

5.12 管理人团队负责人应当制定管理人团队培训制度，明确培训内容、培训方式、培训要求等。

5.13 管理人团队负责人应当制定管理人团队考核制度，明确考核内容、考核方式、考核标准等。

5.14 管理人团队负责人应当制定管理人团队奖惩制度，明确奖励条件、奖励方式、惩罚条件、惩罚方式等。

5.15 管理人团队负责人应当制定管理人团队退出机制，明确退出条件、退出程序、退出责任等。

**第6条 管理人的工作原则**

6.1 律师事务所、律师个人担任管理人，应当勤勉尽责、忠实执行职务，并始终贯彻审慎原则，依法办理相关事务，切实防范法律风险。

6.2 律师事务所、律师个人担任管理人，应当注重工作效率，节约破产费用，减少共益债务。

6.3 律师事务所、律师个人担任管理人，执行管理人职务，应当依法向人民法院报告工作，并依法接受债权人会议和债权人委员会的监督。

6.4 律师事务所、律师个人担任管理人，应当严格履行保密义务。对于在执业中知悉的有关债务人、债权人和其他利害关系人的商业秘密、个人隐私以及其他不能对外披露的事项，管理人团队各成员应当注意保密。

6.5 律师事务所、律师个人担任管理人，应当严格履行廉洁义务，不得利用管理人的身份和职务便利为自己或者他人谋取不正当利益。

6.6 律师事务所、律师个人担任管理人，应当严格履行回避义务，对于与破产案件有利害关系的事项，应当主动回避。

6.7 律师事务所、律师个人担任管理人，应当严格履行报告义务，对于破产案件中的重大事项，应当及时向人民法院报告。

6.8 律师事务所、律师个人担任管理人，应当严格履行披露义务，对于可能影响其独立、公正履行职责的事项，应当及时向人民法院和债权人会议披露。

**第7条 管理人报酬**

7.1 律师事务所、律师个人担任管理人，有权依法获得管理人报酬。管理人报酬由人民法院依照《最高人民法院关于审理企业破产案件确定管理人报酬的规定》确定。

7.2 律师事务所、律师个人担任管理人，应当制定管理人报酬分配制度，明确报酬分配的原则、标准、程序等。

7.3 律师事务所、律师个人担任管理人，应当将管理人报酬的收取和使用情况向人民法院和债权人会议报告。

7.4 律师事务所、律师个人担任管理人，不得违反规定收取管理人报酬，不得以不正当手段获取管理人报酬。

**第8条 管理人工作底稿和档案**

8.1 律师事务所、律师个人担任管理人，应当建立管理人工作底稿制度。管理人工作底稿是管理人履行职责的记录，是管理人工作的重要依据。

8.2 管理人工作底稿应当包括以下内容：

8.2.1 管理人团队组成及分工情况；

8.2.2 管理人工作计划及执行情况；

8.2.3 接管债务人财产、印章、账簿、文书等资料的情况；

8.2.4 调查债务人财产状况的情况；

8.2.5 决定债务人内部管理事务的情况；

8.2.6 决定债务人日常开支和其他必要开支的情况；

8.2.7 决定继续或者停止债务人营业的情况；

8.2.8 管理和处分债务人财产的情况；

8.2.9 代表债务人参加诉讼、仲裁或者其他法律程序的情况；

8.2.10 接收及审查申报债权的情况；

8.2.11 召开债权人会议的情况；

8.2.12 提请裁定宣告破产或终结破产程序的情况；

8.2.13 其他需要记录的情况。

8.3 管理人工作底稿应当真实、完整、准确、清晰，并由制作人签名或者盖章。

8.4 管理人工作底稿应当妥善保管，不得遗失、毁损、篡改。

8.5 管理人终止执行职务后，管理人工作底稿应当移交人民法院或者新指定的管理人。

8.6 律师事务所、律师个人担任管理人，应当建立管理人档案管理制度。管理人档案是管理人履行职责的全部文件资料的总和。

8.7 管理人档案应当包括以下内容：

8.7.1 人民法院指定管理人的法律文书；

8.7.2 管理人团队组成及分工情况；

8.7.3 管理人工作计划及执行情况；

8.7.4 接管债务人财产、印章、账簿、文书等资料的情况；

8.7.5 调查债务人财产状况的报告；

8.7.6 决定债务人内部管理事务的文件；

8.7.7 决定债务人日常开支和其他必要开支的文件；

8.7.8 决定继续或者停止债务人营业的文件；

8.7.9 管理和处分债务人财产的文件；

8.7.10 代表债务人参加诉讼、仲裁或者其他法律程序的文件；

8.7.11 接收及审查申报债权的文件；

8.7.12 召开债权人会议的文件；

8.7.13 提请裁定宣告破产或终结破产程序的文件；

8.7.14 管理人报酬的收取和使用情况；

8.7.15 其他需要归档的文件。

8.8 管理人档案应当真实、完整、准确、清晰，并按照规定整理、装订、编号。

8.9 管理人档案应当妥善保管，不得遗失、毁损、篡改。

8.10 管理人终止执行职务后，管理人档案应当移交人民法院或者新指定的管理人。

### 第二章 裁定受理破产清算申请后的管理人职责

**第9条 一般规定**

9.1 人民法院裁定受理破产清算申请后，管理人应当立即开始履行管理人职责。

9.2 管理人应当制定管理人工作计划，明确各项工作的完成时间、思路、步骤和程序。

9.3 管理人应当及时向人民法院报告工作进展情况。

**第10条 接管债务人**

10.1 管理人应当全面接管债务人，包括但不限于接管债务人的财产、印章、账簿、文书等资料。

10.2 管理人接管债务人财产时，应当制作财产交接清单，由交接双方签字或者盖章确认。

10.3 管理人接管债务人印章时，应当制作印章交接清单，由交接双方签字或者盖章确认。

10.4 管理人接管债务人账簿、文书等资料时，应当制作资料交接清单，由交接双方签字或者盖章确认。

10.5 管理人接管债务人后，应当妥善保管债务人的财产、印章、账簿、文书等资料，不得遗失、毁损、篡改。

**第11条 调查债务人财产状况**

11.1 管理人应当全面调查债务人的财产状况，包括但不限于债务人的资产、负债、所有者权益等。

11.2 管理人调查债务人财产状况时，可以聘请会计师事务所、资产评估机构等专业机构协助。

11.3 管理人调查债务人财产状况后，应当制作财产状况报告，向人民法院和债权人会议报告。

**第12条 决定债务人的内部管理事务**

12.1 管理人应当决定债务人的内部管理事务，包括但不限于债务人的组织机构、人员安排、规章制度等。

12.2 管理人决定债务人的内部管理事务时，应当有利于破产程序的顺利进行，有利于债权人利益的最大化。

**第13条 决定债务人的日常开支和其他必要开支**

13.1 管理人应当决定债务人的日常开支和其他必要开支。

13.2 管理人的日常开支和其他必要开支应当合理、必要，不得浪费。

13.3 管理人应当将日常开支和其他必要开支的情况向人民法院和债权人会议报告。

**第14条 决定继续或者停止债务人的营业**

14.1 在第一次债权人会议召开之前，管理人应当调查债务人的营业情况，形成调查报告，经人民法院许可后，决定继续或者停止债务人的营业。

14.2 第一次债权人会议召开之后，管理人认为有必要继续或者停止债务人营业的，应当提请债权人会议决定。

14.3 管理人决定继续或者停止债务人营业时，应当考虑以下因素：

14.3.1 继续营业是否有利于债务人财产的保值增值；

14.3.2 继续营业是否有利于提高债权人的清偿比例；

14.3.3 继续营业的费用是否合理；

14.3.4 继续营业的风险是否可控；

14.3.5 其他需要考虑的因素。

**第15条 管理和处分债务人的财产**

15.1 管理人应当管理和处分债务人的财产，包括但不限于：

15.1.1 清收债务人的债权；

15.1.2 接受债务人的债务人或者财产持有人的债务清偿或者财产交付；

15.1.3 决定解除或者继续履行破产申请受理前成立而债务人和对方当事人均未履行完毕的合同；

15.1.4 请求人民法院撤销债务人的可撤销行为；

15.1.5 请求人民法院确认债务人的无效行为；

15.1.6 要求债务人的出资人缴纳所认缴的出资；

15.1.7 追回债务人的董事、监事和高级管理人员利用职权从企业获取的非正常收入和侵占的企业财产；

15.1.8 通过清偿债务或者提供为债权人接受的担保，取回质物、留置物；

15.1.9 其他管理和处分债务人财产的行为。

15.2 管理人管理和处分债务人财产时，应当遵循以下原则：

15.2.1 有利于债务人财产的保值增值；

15.2.2 有利于提高债权人的清偿比例；

15.2.3 公平、公正、公开；

15.2.4 依法、合规、审慎。

**第16条 代表债务人参加诉讼、仲裁或者其他法律程序**

16.1 管理人应当代表债务人参加诉讼、仲裁或者其他法律程序。

16.2 管理人代表债务人参加诉讼、仲裁或者其他法律程序时，应当维护债务人的合法权益，不得损害债权人的利益。

**第17条 接收及审查申报债权**

17.1 管理人应当接收债权人的债权申报，登记造册，对申报的债权进行审查，并编制债权表。

17.2 管理人接收债权申报时，应当向债权人出具接收凭证。

17.3 管理人审查债权时，应当审查债权的真实性、合法性、有效性。

17.4 管理人编制债权表后，应当提交第一次债权人会议核查。

**第18条 召开债权人会议**

18.1 管理人应当协助人民法院召集第一次债权人会议。

18.2 管理人应当于债权人会议召开十五日前通知已知债权人。

18.3 管理人应当列席债权人会议，向债权人会议报告职务执行情况，并回答询问。

**第19条 提请裁定宣告破产或终结破产程序**

19.1 管理人认为债务人符合宣告破产条件的，应当提请人民法院裁定宣告债务人破产。

19.2 管理人认为债务人财产不足以清偿破产费用的，应当提请人民法院裁定终结破产程序。

19.3 管理人认为债务人无财产可供分配的，应当提请人民法院裁定终结破产程序。

19.4 管理人认为破产财产分配完毕的，应当提请人民法院裁定终结破产程序。

### 第三章 裁定重整后的管理人职责

**第20条 一般规定**

20.1 人民法院裁定债务人重整后，管理人应当继续履行管理人职责，但管理人职责的内容和方式可能发生变化。

20.2 在重整期间，经债务人申请，人民法院批准，债务人可以在管理人的监督下自行管理财产和营业事务。

20.3 债务人自行管理财产和营业事务的，管理人应当履行监督职责。

**第21条 管理人管理债务人财产和营业事务**

21.1 债务人未自行管理财产和营业事务的，管理人应当管理债务人的财产和营业事务。

21.2 管理人管理债务人财产和营业事务时，应当制定重整计划草案，并提交人民法院和债权人会议。

21.3 管理人制定重整计划草案时，应当充分听取债权人、债务人、出资人等各方的意见。

21.4 管理人制定重整计划草案时，应当考虑以下因素：

21.4.1 债务人的资产状况和经营能力；

21.4.2 债权人的利益保护；

21.4.3 债务人的职工安置；

21.4.4 债务人的社会责任；

21.4.5 其他需要考虑的因素。

**第22条 管理人监督债务人管理财产和营业事务**

22.1 债务人自行管理财产和营业事务的，管理人应当履行监督职责。

22.2 管理人监督债务人管理财产和营业事务时，应当制定监督方案，明确监督的内容、方式、程序等。

22.3 管理人发现债务人有损害债权人利益或者其他违法行为的，应当及时向人民法院报告，并请求人民法院作出处理。

**第23条 重整计划的制定、通过与批准**

23.1 管理人或者债务人应当自人民法院裁定债务人重整之日起六个月内，同时向人民法院和债权人会议提交重整计划草案。

23.2 管理人或者债务人提交重整计划草案的期限届满，经管理人或者债务人请求，有正当理由的，人民法院可以裁定延期三个月。

23.3 管理人或者债务人未按期提出重整计划草案的，人民法院应当裁定终止重整程序，并宣告债务人破产。

23.4 重整计划草案应当包括以下内容：

23.4.1 债务人的经营方案；

23.4.2 债权分类；

23.4.3 债权调整方案；

23.4.4 债权受偿方案；

23.4.5 重整计划的执行期限；

23.4.6 重整计划执行的监督期限；

23.4.7 有利于债务人重整的其他方案。

23.5 重整计划草案应当分组表决，各表决组均通过重整计划草案时，重整计划即为通过。

23.6 部分表决组未通过重整计划草案的，管理人或者债务人可以同未通过重整计划草案的表决组协商。该表决组可以在协商后再表决一次。

23.7 未通过重整计划草案的表决组拒绝再次表决或者再次表决仍未通过重整计划草案，但重整计划草案符合法定条件的，管理人或者债务人可以申请人民法院批准重整计划草案。

23.8 重整计划草案未获得通过且未依照前款规定获得人民法院批准的，或者已通过的重整计划未获得人民法院批准的，人民法院应当裁定终止重整程序，并宣告债务人破产。

**第24条 重整计划的执行**

24.1 重整计划由债务人负责执行。

24.2 管理人应当监督重整计划的执行。

24.3 管理人监督重整计划执行时，应当制定监督方案，明确监督的内容、方式、程序等。

24.4 管理人发现债务人不执行或者不能执行重整计划的，应当及时向人民法院报告，并请求人民法院裁定终止重整计划的执行，并宣告债务人破产。

24.5 重整计划执行完毕后，管理人应当向人民法院提交监督报告，并申请人民法院裁定终结重整程序。

### 第四章 裁定和解后的管理人职责

**第25条 一般规定**

25.1 人民法院裁定债务人和解后，管理人应当继续履行管理人职责，但管理人职责的内容和方式可能发生变化。

25.2 在和解期间，债务人可以在管理人的监督下管理财产和营业事务。

**第26条 和解协议的通过**

26.1 管理人应当协助债务人和债权人就和解协议进行协商。

26.2 管理人应当协助人民法院审查和解协议的内容。

26.3 和解协议经债权人会议通过并经人民法院裁定认可后，对债务人和全体和解债权人均有约束力。

**第27条 和解协议的执行**

27.1 和解协议由债务人负责执行。

27.2 管理人应当监督和解协议的执行。

27.3 管理人发现债务人不执行或者不能执行和解协议的，应当及时向人民法院报告，并请求人民法院裁定终止和解协议的执行，并宣告债务人破产。

27.4 和解协议执行完毕后，管理人应当向人民法院提交监督报告，并申请人民法院裁定终结和解程序。

### 第五章 宣告破产后的管理人职责

**第28条 一般规定**

28.1 人民法院宣告债务人破产后，管理人应当继续履行管理人职责，直至破产程序终结。

28.2 管理人应当制定破产财产变价方案，提交债权人会议讨论。

28.3 管理人应当制定破产财产分配方案，提交债权人会议讨论。

**第29条 破产财产变价方案**

29.1 管理人应当及时拟订破产财产变价方案，提交债权人会议讨论。

29.2 破产财产变价方案应当包括以下内容：

29.2.1 变价的财产范围；

29.2.2 变价的方式；

29.2.3 变价的时间安排；

29.2.4 变价的底价；

29.2.5 其他需要说明的事项。

29.3 破产财产变价方案经债权人会议通过后，管理人应当按照方案进行变价。

**第30条 破产财产分配方案**

30.1 管理人应当及时拟订破产财产分配方案，提交债权人会议讨论。

30.2 破产财产分配方案应当包括以下内容：

30.2.1 参加财产分配的债权人名称或者姓名、住所；

30.2.2 参加财产分配的债权额；

30.2.3 可供分配的财产数额；

30.2.4 财产分配的顺序、比例及数额；

30.2.5 实施财产分配的方法；

30.2.6 其他需要说明的事项。

30.3 破产财产分配方案经债权人会议通过后，管理人应当提请人民法院裁定认可。

30.4 人民法院裁定认可破产财产分配方案后，管理人应当按照方案进行分配。

**第31条 破产程序的终结**

31.1 管理人认为破产财产分配完毕的，应当提请人民法院裁定终结破产程序。

31.2 管理人认为债务人无财产可供分配的，应当提请人民法院裁定终结破产程序。

31.3 管理人认为债务人财产不足以清偿破产费用的，应当提请人民法院裁定终结破产程序。

31.4 人民法院裁定终结破产程序的，管理人应当向债务人的登记机关办理注销登记。

31.5 人民法院裁定终结破产程序的，管理人应当终止执行职务。

### 第六章 管理人终止执行职务

**第32条 一般规定**

32.1 管理人应当在人民法院裁定终结破产程序后终止执行职务。

32.2 管理人终止执行职务前，应当向人民法院提交执行职务的报告。

32.3 管理人终止执行职务前，应当将其管理的全部资料、财产、印章和账簿、文书等移交人民法院或者新指定的管理人。

32.4 管理人终止执行职务后，其管理人职责即告终止。

**第33条 执行职务报告**

33.1 管理人终止执行职务前，应当向人民法院提交执行职务的报告。

33.2 执行职务的报告应当包括以下内容：

33.2.1 管理人履行职责的情况；

33.2.2 债务人财产的管理和处分情况；

33.2.3 债权申报和审查情况；

33.2.4 债权人会议召开情况；

33.2.5 破产财产变价和分配情况；

33.2.6 管理人报酬的收取和使用情况；

33.2.7 其他需要说明的事项。

**第34条 移交**

34.1 管理人终止执行职务前，应当将其管理的全部资料、财产、印章和账簿、文书等移交人民法院或者新指定的管理人。

34.2 管理人移交时，应当制作移交清单，由交接双方签字或者盖章确认。

34.3 管理人移交后，应当协助新指定的管理人熟悉情况，提供必要的协助。

## 核心要点

### 管理人回避情形

| 类型 | 具体情形 |
|------|----------|
| 绝对回避 | 故意犯罪受过刑事处罚；曾被吊销执业证书；与案件有利害关系 |
| 律所利害关系 | 与债务人/债权人有未了结债权债务；受理前 3 年内提供固定法律服务；曾为控股股东/实际控制人；曾担任财务/法律顾问 |
| 律师个人利害关系 | 具有律所利害关系情形；曾担任董监高；与控股股东/董监高存在近亲属关系 |
| 不宜担任情形 | 受行政处罚未逾 3 年；正被调查；执业证被吊销；被除名未逾 3 年；缺乏专业能力；缺乏民事责任能力；重大债务纠纷 |

### 管理人团队职责分工

| 职责 | 内容 |
|------|------|
| 团队负责人 | 确定分工、制定计划、联络法院、对外签约、提交报告、掌管印章、审批费用、召集会议 |
| 接管组 | 接管财产、印章、账簿、文书等 |
| 调查组 | 调查财产状况、制作财产报告 |
| 债权组 | 接收审查债权、编制债权表 |
| 资产组 | 管理和处分债务人财产 |
| 诉讼组 | 代表债务人参加诉讼、仲裁 |
| 会议组 | 协助召集债权人会议 |

### 管理人工作原则

- 勤勉尽责、忠实执行职务
- 审慎原则，防范法律风险
- 注重效率，节约费用，减少共益债务
- 严格保密义务
- 严格廉洁义务
- 严格回避义务
- 严格报告义务
- 严格披露义务

### 与《企业破产法》的衔接

- 管理人指定、回避、辞职和更换的具体情形
- 管理人团队的组建和职责分工
- 管理人工作底稿和档案管理制度
- 破产清算、重整、和解各阶段的管理人职责

### 与司法解释的衔接

本指引与《最高人民法院关于审理企业破产案件指定管理人的规定》《最高人民法院关于审理企业破产案件确定管理人报酬的规定》等司法解释相衔接，为律师担任管理人提供了全面的操作规范。
