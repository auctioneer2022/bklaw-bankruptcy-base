# 管理人规范

## 概述

管理人规范是指导破产管理人依法、规范履行职责的重要文件，包括全国性管理人协会规范、地方性管理人协会规范以及管理人工作指引等。这些规范为管理人提供了操作指南，有助于提高管理人履职能力，保障破产程序的顺利进行。
