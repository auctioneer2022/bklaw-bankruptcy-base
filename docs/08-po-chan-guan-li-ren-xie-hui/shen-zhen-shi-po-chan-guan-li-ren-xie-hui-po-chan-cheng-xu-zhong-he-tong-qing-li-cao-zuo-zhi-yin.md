# 深圳市破产管理人协会破产程序中合同清理操作指引（试行）

> **发文机关**：深圳市破产管理人协会
> **通过日期**：2024年11月21日（经深圳市破产管理人协会第二届理事会临时会议表决通过）
> **时效性**：试行

## 文书要点

《深圳市破产管理人协会破产程序中合同清理操作指引（试行）》是深圳市破产管理人协会于2024年11月21日制定发布的行业自律规范，旨在进一步发挥破产制度的市场主体救济与退出功能，提升管理人履职能力，优化营商环境。

该指引围绕破产程序中**合同清理**这一管理人核心职责，从**未履行完毕合同的处理方式**、**继续履行或解除的具体情形**、**撤销权的行使**三个维度，为管理人提供了系统性的操作规范。

### 核心要点

| 事项 | 要点 |
|------|------|
| 基本原则 | 有利于财产保值增值、提高债权人清偿比例 |
| 法律依据 | 《企业破产法》第十八条、《民法典》合同编相关规定 |
| 报告程序 | 第一次债权人会议前报法院许可；之后报债权人委员会（或法院） |
| 特殊合同 | 买卖合同（所有权保留）、供用电水气热力合同、赠与合同 |
| 撤销权 | 破产撤销权与民事撤销权竞合时的处理规则 |
| 明显不合理价格 | 低于市场价70%为明显不合理低价；高于市场价30%为明显不合理高价 |

## 全文

### 一、未履行完毕合同的处理方式

**（一）基本原则。** 人民法院受理破产申请后，管理人对破产申请受理前成立而债务人和对方当事人均未履行完毕的合同应及时进行清理，以有利于财产保值增值、提高债权人清偿比例为原则，拒绝执行现有负担过重的合同和资产，避免优惠交易和低于一般价值的交易，依法决定解除或者继续履行。

**（二）法律依据。** 管理人应当依据《企业破产法》第十八条以及《民法典》第五百六十二条、九百三十三条、五百六十三条及其他法律有关约定解除权、法定解除权、任意解除权的相关规定，决定是否解除未履行完毕的合同。

**（三）法律后果。** 管理人决定继续履行合同，对方当事人有权要求管理人提供继续履行合同的担保；管理人不提供担保的，视为解除合同。管理人或者债务人根据《企业破产法》规定解除合同，因合同解除而产生的对方当事人对债务人的债务的，或者因合同解除对方当事人应当向债务人交付财产的，管理人应当在合同解除后要求对方当事人清偿债务或者交付财产。

**（四）报告人民法院。** 第一次债权人会议召开之前，对于在破产申请受理前成立而债务人和对方当事人均未履行完毕的合同，管理人认为有必要继续履行的，应当经人民法院许可。

**（五）报告债权人委员会。** 第一次债权人会议召开之后，对于在破产申请受理前成立而债务人和对方当事人均未履行完毕的合同，管理人认为有必要继续履行的，管理人应当及时报告债权人委员会；未设立债权人委员会的，管理人应当及时报告人民法院。

### 二、继续履行或者解除未履行完毕合同的具体情形

**（六）买卖合同。**

**出卖人破产**的，管理人决定继续履行所有权保留买卖合同的，可以请求买受人按照原买卖合同的约定支付价款或者履行其他义务。管理人决定解除所有权保留买卖合同的，可以依据《企业破产法》第十七条的规定要求买受人交付买卖标的物。

**买受人破产**的，管理人决定继续履行所有权保留买卖合同的，原买卖合同中约定的买受人支付价款或者履行其他义务的期限在破产申请受理时视为到期，管理人应当及时向出卖人支付价款或者履行其他义务。管理人决定解除所有权保留买卖合同的，出卖人可以依据《企业破产法》第三十八条的规定主张取回买卖标的物。

**（七）供用电、水、气、热力合同。** 债务人作为用电、用水、用气、用热单位签订的供用电、供用水、供用气、供用热力合同，管理人应当根据债务人实际营业状况决定是否解除。

**（八）赠与合同。** 债务人作为赠与人签订的赠与合同，在破产申请受理时未履行完毕，管理人可以不再继续履行。

### 三、撤销权的行使

**（九）可撤销行为。** 管理人发现债务人存在《企业破产法》第三十一条、第三十二条规定行为的，应当及时请求人民法院予以撤销。但是，个别清偿使债务人财产受益的除外。

**（十）民事撤销权和破产撤销权的竞合。** 人民法院受理破产申请后，如果管理人无法依据《企业破产法》第三十一条、第三十二条规定行使撤销权，管理人可以代表债务人根据《民法典》关于可撤销法律行为的规定请求撤销并追回财产。

**（十一）明显不合理的价格。** 对明显不合理价格的认定，管理人可以参考交易时交易地物价部门指导价或者市场交易价并结合其他相关因素综合考虑予以认定。

转让价格达不到交易时交易地的指导价或者市场交易价百分之七十的，一般可以视为明显不合理的低价；对转让价格高于当地指导价或者市场交易价百分之三十的，一般可以视为明显不合理的高价。债务人、相对人可以向管理人提交证据证明其所主张的交易时交易地的指导价或者市场交易价。

**（十二）债权人撤销权。** 人民法院受理破产申请后，管理人未依据《企业破产法》第三十一条的规定请求撤销债务人以明显不合理价格交易行为的，债权人可以依据《民法典》第五百三十八条、第五百三十九条等规定提起诉讼，请求撤销债务人上述行为并将因此追回的财产归入债务人财产。

## 核心要点

### 未履行完毕合同的处理流程

```
受理破产申请
    ↓
管理人及时清理未履行完毕合同
    ↓
判断：继续履行 or 解除
    ↓
第一次债权人会议前 → 报人民法院许可
第一次债权人会议后 → 报债权人委员会（未设立的报法院）
    ↓
继续履行：对方有权要求提供担保；不提供则视为解除
解除合同：要求对方清偿债务或交付财产
```

### 特殊合同处理规则

| 合同类型 | 处理规则 |
|----------|----------|
| 所有权保留买卖合同（出卖人破产） | 继续履行：请求买受人支付价款；解除：要求交付标的物 |
| 所有权保留买卖合同（买受人破产） | 继续履行：付款期限视为到期；解除：出卖人可主张取回 |
| 供用电、水、气、热力合同 | 根据实际营业状况决定是否解除 |
| 赠与合同 | 未履行完毕的，管理人可不再继续履行 |

### 撤销权行使规则

| 撤销权类型 | 适用情形 | 法律依据 |
|------------|----------|----------|
| 破产撤销权 | 《企业破产法》第31条、第32条规定的行为 | 《企业破产法》第31-32条 |
| 民事撤销权 | 无法依据破产撤销权行使时 | 《民法典》关于可撤销法律行为的规定 |
| 债权人撤销权 | 管理人未行使破产撤销权时 | 《民法典》第538条、第539条 |

### 明显不合理价格的认定标准

| 类型 | 认定标准 |
|------|----------|
| 明显不合理低价 | 转让价格 < 市场价/指导价的70% |
| 明显不合理高价 | 转让价格 > 市场价/指导价的130% |

## 与破产实务的关联

### 管理人挑拣履行权的行使要点

1. **及时性**：管理人应在破产申请受理后及时对未履行完毕合同进行清理，避免合同长期处于不确定状态
2. **利益最大化原则**：以有利于财产保值增值、提高债权人清偿比例为根本判断标准
3. **报告义务**：根据时间节点区分报告对象（法院/债权人委员会）
4. **担保义务**：决定继续履行的，应准备提供担保；不能提供的，视为解除合同

### 与《企业破产法》第十八条的关系

《企业破产法》第十八条规定了管理人对未履行完毕合同的挑拣履行权，本指引在此基础上进一步细化了：
- 具体操作流程（报告程序、时间节点）
- 特殊合同类型的处理规则
- 撤销权竞合时的处理方案
- 明显不合理价格的量化认定标准

### 与《民法典》的衔接

本指引明确管理人行使合同解除权时，除适用《企业破产法》外，还应结合《民法典》关于约定解除权、法定解除权、任意解除权的相关规定，实现了破产法与民法典的有机衔接。
