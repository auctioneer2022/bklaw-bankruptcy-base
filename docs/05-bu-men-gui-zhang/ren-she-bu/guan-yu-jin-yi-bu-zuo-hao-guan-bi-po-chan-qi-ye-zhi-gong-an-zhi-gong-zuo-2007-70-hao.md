# 关于进一步做好关闭破产企业职工安置工作有关问题的通知

**文号**：川劳社办〔2007〕70号  
**发布机关**：四川省劳动和社会保障厅  
**发布日期**：2007年9月20日  
**施行日期**：2007年9月20日  
**效力状态**：现行有效（历史政策文件，具有参考价值）

---

## 文书要点

本通知是四川省劳动和社会保障厅根据劳动保障部全国部分省市关闭破产企业职工安置工作座谈会精神，结合四川省实际制定的重要文件，对关闭破产企业职工安置工作的政策适用、方案审核、工作职责、维稳措施等作出了具体规定，是四川省破产企业职工安置工作的重要指导性文件。

## 通知全文

各市（州）劳动和社会保障局：

最近，劳动保障部召开了全国部分省市劳动保障部门和中央企业集团关闭破产企业职工安置工作座谈会。现将会议主要精神通报如下，同时就进一步做好我省关闭破产企业职工安置工作提出以下意见，请各地及时组织学习传达，并认真按此执行。

### 一、会议主要精神

**（一）认清形势，在努力维护稳定大局的前提下继续推进国有企业深化改革**

在破产安置和并轨中，有四大关系需要在思想上进一步明确，在工作中更好的把握。

一是维护大局和谐稳定与坚持改革不停步的关系。在党的十七大召开至明年"两会"这一大的背景下，如何通过我们的工作，维护企业和社会的稳定，为开好大会，办好大事创造和谐稳定的社会环境，是必须牢牢把握的大局。与此同时，深化改革的任务不能停步。关闭破产工作是深化国有企业改革重要内容和关键战役，必须坚定不移地继续推进。

二是加快审批进度与把握实施节奏、做实做细安置工作的关系。今明两年，是国有企业关闭破产的关键期，全国领导小组已经完成了总体规划内需要实施破产的最后700余户项目的审核工作，要求在今年年底到明年基本完成项目的组织实施，这就对我们做好职工安置工作提出了重要挑战。一定要做到，越是在工作进度加快的情况下，越要认真把握好职工安置工作的节奏，安置方案的审核不能放松，政策宣传和思想工作不能忽视，经济补偿、社会保险接续、职工再就业等方面的工作必须做实做细。

三是用政策治本与用工作措施治标的关系。关闭破产企业职工安置工作积累的一些问题，涉及到大的政策调整和资金增加的问题，所以还要加强政策研究，解决资金筹措，努力从根本上找到解决问题的办法。但大的政策调整和资金问题的解决不是一下子就能到位的情况下，尤其要防止政策调整可能会引起连锁反应和"翻烧饼"的问题。要坚持标本兼治，把握好从根本上解决问题的政策措施与现行政策框架下做好落实和服务的关系，不能只强调一方面而忽视另一方面。

四是解决当前突出矛盾与解决潜在矛盾之间的关系。当前，关闭破产企业职工安置工作的矛盾和问题已很明显，我们要重视解决。同时对并轨工作在面上已基本完成，但遗留问题尚多，不能盲目乐观和松懈，更不能造成领导的错觉。对节能减排等工作力度加大将带来的新的失业问题也要保持关注和警惕，做好相应的预警，防患于未然。

在分析我们工作面临的困难和挑战的同时，也应该看到做好工作的有利条件：党中央、国务院对这项工作高度重视，进一步提出了明确要求；新的《破产法》已经开始实施，《劳动合同法》、《就业促进法》相继出台，为做好工作提供了强有力的法律支持；我国经济持续快速发展，解决遗留问题有了更加雄厚的经济基础；工作中摸索出了一些行之有效的经验做法，可以继续推广和借鉴。

**（二）在积极推进关闭破产企业职工安置和并轨工作中，要千方百计确保社会稳定**

今明两年维护社会和谐与大局稳定是第一位的任务。在这期间，各地和中央企业集团要按照中央、国务院的部署和要求，在关闭破产企业职工安置和处理并轨遗留问题这两个方面，紧紧围绕稳定这个主题，全力以赴，扎实平稳地推进工作，坚持标本兼治，采取有效措施，抓好主要环节，化解突出矛盾，力争不发生群体性事件，为党的十七大胜利召开和贯彻落实十七大精神以及明年的"两会"和奥运会创造良好的社会环境。具体来说，下一步要着力做好以下工作：

1. **广泛动员，提高认识**。各地和中央企业集团要进一步提高认识，增强责任感、使命感，要把维护稳定这一指导思想贯穿于工作的始终。进一步增强保稳定、服务大局的意识。

2. **明确责任，抓好落实**。各级劳动保障部门和中央企业集团要建立维护稳定的指挥体系、组织体系和工作体系。主要负责人是第一责任人，负总责。对于每个关闭破产项目，都要有专人负责，并要实行目标责任制，层层落实，责任到人。中央企业集团是所属破产企业职工安置工作的责任主体，要切实负起责任。

3. **加强排查，化解矛盾**。要建立矛盾纠纷排查调处机制，及时发现和化解矛盾。对职工反映强烈的问题，要认真研究，及时解决。对可能引发群体性事件的苗头，要早发现、早报告、早处置。

4. **完善预案，妥善处置**。要制定和完善群体性事件应急预案，明确处置程序和责任分工。一旦发生群体性事件，要迅速启动预案，妥善处置，防止事态扩大。

5. **加强沟通，形成合力**。要加强与有关部门的沟通协调，形成工作合力。要及时向党委、政府报告工作进展情况，争取党委、政府的重视和支持。

### 二、进一步做好我省关闭破产企业职工安置工作的意见

**（一）严格执行并准确适用政策**

各地在指导关闭破产企业做好职工安置工作中，要严格执行并准确适用政策。要严格审核职工安置方案，对职代会通过率较低、资金未落实、程序不规范、再就业措施不落实、医疗保险问题没有明确解决办法的职工安置方案，要重点把关，并督促企业予以调整改正。同时，要善于掌握审核时机，稳妥批复职工安置方案。

**（二）进一步明确工作职责**

各地劳动保障部门要在当地党委、政府的领导下，积极履行职责，主动做好关闭破产企业职工安置工作。要加强与国资、经贸、财政、工会等部门的协调配合，形成工作合力。要明确工作职责，落实工作责任，确保各项工作落到实处。

**（三）切实做好职工安置方案的审核工作**

职工安置方案是关闭破产企业职工安置工作的核心。各地劳动保障部门要切实做好职工安置方案的审核工作，重点审核以下内容：

1. 职工安置方案是否经职工代表大会或全体职工大会讨论通过；
2. 职工安置所需资金是否落实；
3. 职工经济补偿标准是否符合国家和省有关规定；
4. 职工社会保险关系接续方案是否明确；
5. 职工再就业措施是否落实；
6. 职工医疗保险问题是否有明确解决办法；
7. 其他需要审核的内容。

**（四）加强职工安置资金的监督管理**

职工安置资金是做好职工安置工作的物质基础。各地劳动保障部门要加强对职工安置资金的监督管理，确保资金专款专用，及时足额发放到职工手中。要防止资金被挪用、截留、克扣等问题发生。

**（五）做好职工社会保险关系接续工作**

各地劳动保障部门要积极做好关闭破产企业职工社会保险关系接续工作，确保职工社会保险权益不受损害。要指导企业及时为职工办理社会保险关系转移手续，确保职工社会保险关系顺畅接续。

**（六）积极促进职工再就业**

各地劳动保障部门要将关闭破产企业职工再就业纳入当地就业再就业工作总体安排，积极采取措施，促进职工再就业。要落实各项再就业扶持政策，为职工提供职业指导、职业培训、职业介绍等就业服务。要开发就业岗位，拓宽就业渠道，帮助职工尽快实现再就业。

**（七）切实维护职工合法权益**

各地劳动保障部门要切实维护关闭破产企业职工合法权益，对侵害职工合法权益的行为，要依法予以纠正。要畅通职工诉求表达渠道，及时受理职工投诉举报，依法处理劳动争议。要做好职工思想政治工作，引导职工依法理性表达诉求。

**（八）加强信息报送和舆情监测**

各地劳动保障部门要加强关闭破产企业职工安置工作信息报送，及时向上级部门报告工作进展情况和存在的问题。要加强舆情监测，及时掌握社会反映，对涉及关闭破产企业职工安置的负面舆情，要及时应对，正确引导。

## 核心要点

### 1. 职工安置方案审核重点

| 审核项目 | 审核内容 |
|---------|---------|
| 民主程序 | 是否经职工代表大会或全体职工大会讨论通过 |
| 资金落实 | 职工安置所需资金是否落实 |
| 补偿标准 | 职工经济补偿标准是否符合国家和省有关规定 |
| 社保接续 | 职工社会保险关系接续方案是否明确 |
| 再就业措施 | 职工再就业措施是否落实 |
| 医疗保险 | 职工医疗保险问题是否有明确解决办法 |

### 2. 四大关系把握

| 关系 | 核心要义 |
|------|---------|
| 稳定与改革 | 维护稳定大局与坚持改革不停步 |
| 进度与质量 | 加快审批进度与做实做细安置工作 |
| 治标与治本 | 用政策治本与用工作措施治标 |
| 当前与潜在 | 解决当前突出矛盾与解决潜在矛盾 |

### 3. 维稳工作要求

- 建立维护稳定的指挥体系、组织体系和工作体系
- 主要负责人是第一责任人，实行目标责任制
- 建立矛盾纠纷排查调处机制
- 制定和完善群体性事件应急预案
- 加强与有关部门的沟通协调

## 与破产实务的关联

### 1. 政策性破产的历史背景

本通知发布于2007年，正值国有企业政策性关闭破产的关键时期。当时全国领导小组已完成总体规划内最后700余户项目的审核工作，要求在2007年底到2008年基本完成组织实施。这一历史背景对于理解通知的内容和精神具有重要意义。

### 2. 与现行《企业破产法》的衔接

虽然本通知主要针对政策性破产，但其中的一些原则和精神对现行《企业破产法》下的职工安置工作仍具有参考价值：
- 职工安置方案的严格审核
- 职工合法权益的维护
- 社会保险关系的接续
- 职工再就业的促进
- 维稳工作的重视

### 3. 对四川破产案件的指导意义

作为四川省劳动和社会保障厅发布的文件，本通知对四川省内的破产案件职工安置工作具有直接的指导意义。管理人、法院在处理四川省内破产案件时，应参考本通知的规定，做好职工安置工作。

### 4. 职工安置方案审核的借鉴

本通知明确的职工安置方案审核重点，对现行破产程序中管理人制定职工安置方案具有借鉴意义：
- 确保安置方案经过民主程序
- 确保安置资金落实到位
- 确保补偿标准符合规定
- 确保社保接续方案明确
- 确保再就业措施具体可行
