## 内容说明

中国证监会及其下属证券交易所发布的上市公司破产重整相关规定，是规范上市公司破产重整程序、保护投资者合法权益、维护证券市场秩序的重要制度依据。这些规定与《企业破产法》、最高人民法院相关司法解释及座谈会纪要共同构成上市公司破产重整的法律规范体系。
