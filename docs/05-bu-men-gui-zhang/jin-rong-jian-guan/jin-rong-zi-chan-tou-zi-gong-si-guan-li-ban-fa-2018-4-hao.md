# 金融资产投资公司管理办法（试行）

**文号**：银保监会令 2018 年第 4 号
**发布机关**：中国银行保险监督管理委员会
**发布日期**：2018 年 6 月 29 日
**施行日期**：2018 年 6 月 29 日
**效力状态**：现行有效

---

## 文书要点

本办法是规范金融资产投资公司（AIC）设立和运营的重要规章，确立了市场化、法治化银行债权转股权的制度框架。金融资产投资公司作为实施债转股的专业机构，在企业破产重整中扮演着重要角色，可以通过收购银行债权、转为股权等方式参与重整。

## 办法全文（核心章节）

### 第一章 总则

**第一条** 为推动市场化、法治化银行债权转股权健康有序开展，规范银行债权转股权（以下简称债转股）业务行为，根据《中华人民共和国银行业监督管理法》《中华人民共和国商业银行法》和《中华人民共和国公司法》等法律法规以及《国务院关于积极稳妥降低企业杠杆率的意见》《中国人民银行 中国银行保险监督管理委员会 中国证券监督管理委员会 国家外汇管理局关于规范金融机构资产管理业务的指导意见》，制定本办法。

**第二条** 本办法所称金融资产投资公司是指经国务院银行业监督管理机构批准，在中华人民共和国境内设立的，主要从事银行债权转股权及配套支持业务的非银行金融机构。

**第三条** 金融资产投资公司应当遵循市场化、法治化原则运作，与各参与主体在依法合规前提下，通过自愿平等协商开展债转股业务，确保洁净转让、真实出售，坚持通过市场机制发现合理价格，切实防止企业风险向银行业金融机构和社会公众转移，防止利益冲突和利益输送，防范相关道德风险。

**第四条** 银行通过金融资产投资公司实施债转股，应当通过向金融资产投资公司转让债权，由金融资产投资公司将债权转为对象企业股权的方式实现。银行不得直接将债权转化为股权，但国家另有规定的除外。

鼓励金融资产投资公司通过先收购银行对企业的债权，再将债权转为股权的形式实施债转股，收购价格由双方按市场化原则自主协商确定。涉及银行不良资产，可以按不良资产处置的有关规定办理。鼓励银行及时利用已计提拨备核销资产转让损失。

**第五条** 银行、金融资产投资公司应当与债转股对象企业、企业股东等相关方按照公允原则确定股权数量和价格，依法建立合理的损失分担机制，真实降低企业杠杆率，切实化解金融风险。

鼓励通过债转股、原股东资本减记、引进新股东等方式优化企业股权结构。支持金融资产投资公司推动企业改组改制，切实行使股东权利，履行股东义务，提高企业公司治理水平。

### 第二章 设立、变更与终止

**第七条** 金融资产投资公司应当具备下列条件：

（一）有符合《中华人民共和国公司法》和国务院银行业监督管理机构规定的章程；

（二）有符合本办法要求的股东和注册资本；

（三）有符合任职资格条件的董事、高级管理人员和熟悉业务的合格从业人员；

（四）建立有效的公司治理、内部控制和风险管理制度，有与业务经营相适应的信息科技系统；

（五）有与业务经营相适应的营业场所、安全防范措施和其他设施；

（六）国务院银行业监督管理机构规章规定的其他审慎性条件。

**第八条** 金融资产投资公司应当由在中华人民共和国境内注册成立的商业银行作为主要股东发起设立。

### 第三章 业务范围和业务规则

**第十七条** 金融资产投资公司开展债转股业务，应当遵循以下原则：

（一）市场化原则。银行、金融资产投资公司、债转股对象企业等市场主体自主协商确定债转股的价格、条件、期限等具体事项，自担风险、自享收益。

（二）法治化原则。债转股业务应当在法律法规允许的范围内开展，依法保护各参与主体的合法权益。

（三）审慎性原则。金融资产投资公司应当建立有效的风险管理和内部控制机制，审慎评估债转股业务的风险，确保业务稳健开展。

**第十八条** 金融资产投资公司可以开展以下业务：

（一）以债转股为目的收购银行对企业的债权，将债权转为股权并对股权进行管理；

（二）对于未能转股的债权进行重组、转让和处置；

（三）以债转股为目的投资企业股权，由企业将股权投资资金全部用于偿还现有债权；

（四）依法依规面向合格投资者募集资金，发行私募资产管理产品支持实施债转股；

（五）发行金融债券；

（六）提供与债转股相关的财务顾问、咨询等服务；

（七）经国务院银行业监督管理机构批准的其他业务。

**第十九条** 金融资产投资公司不得从事以下业务：

（一）吸收公众存款；

（二）发放贷款；

（三）受托发放贷款；

（四）受托投资；

（五）与债转股无关的证券、期货等金融投资；

（六）法律法规规定不得从事的其他业务。

**第二十条** 金融资产投资公司收购银行债权，应当遵守以下规定：

（一）收购的债权应当是真实、合法、有效的；

（二）收购价格应当由双方按市场化原则自主协商确定；

（三）涉及不良资产的，可以按照不良资产处置的有关规定办理；

（四）银行应当及时利用已计提拨备核销资产转让损失。

**第二十一条** 金融资产投资公司将债权转为股权，应当遵守以下规定：

（一）转股对象企业应当具备发展前景良好、技术先进、市场认可度高、社会信誉良好等条件；

（二）转股价格应当由相关各方按照市场化原则协商确定；

（三）转股后，金融资产投资公司应当依法行使股东权利，参与公司治理；

（四）转股后，金融资产投资公司可以通过上市、股权转让、企业回购等方式实现股权退出。

### 第四章 风险管理

**第二十五条** 金融资产投资公司应当建立全面风险管理体系，有效识别、计量、监测和控制各类风险。

**第二十六条** 金融资产投资公司应当建立完善的内部控制制度，明确各部门、各岗位的职责权限，建立有效的制衡机制。

**第二十七条** 金融资产投资公司应当建立科学的投资决策机制，对债转股项目进行充分的尽职调查和风险评估。

**第二十八条** 金融资产投资公司应当建立股权管理制度，对持有的股权进行持续管理，及时了解被投资企业的经营状况和财务状况，防范股权风险。

**第二十九条** 金融资产投资公司应当建立风险准备金制度，按照有关规定提取风险准备金。

### 第五章 监督管理

**第三十一条** 国务院银行业监督管理机构及其派出机构依法对金融资产投资公司及其业务活动实施监督管理。

**第三十二条** 金融资产投资公司应当按照有关规定向国务院银行业监督管理机构报送财务会计报告、统计报表、风险管理报告等材料。

## 核心要点

### 1. 金融资产投资公司定义与定位

| 项目 | 内容 |
|------|------|
| 性质 | 非银行金融机构 |
| 主营业务 | 银行债权转股权及配套支持业务 |
| 发起设立 | 由商业银行作为主要股东发起设立 |
| 运作原则 | 市场化、法治化 |

### 2. 债转股业务模式

| 模式 | 说明 |
|------|------|
| 收购债权再转股 | 先收购银行对企业的债权，再将债权转为股权（鼓励模式） |
| 直接投资股权 | 以债转股为目的投资企业股权，由企业将资金用于偿还现有债权 |
| 债权重组处置 | 对未能转股的债权进行重组、转让和处置 |

### 3. 业务范围

| 允许业务 | 禁止业务 |
|---------|---------|
| 收购银行债权并转股 | 吸收公众存款 |
| 债权重组、转让和处置 | 发放贷款 |
| 投资企业股权 | 受托发放贷款 |
| 发行私募资产管理产品 | 受托投资 |
| 发行金融债券 | 与债转股无关的证券、期货投资 |
| 提供财务顾问、咨询服务 | — |

### 4. 风险管理要求

| 要求 | 内容 |
|------|------|
| 全面风险管理 | 有效识别、计量、监测和控制各类风险 |
| 内部控制 | 明确职责权限，建立有效制衡机制 |
| 投资决策 | 充分尽职调查和风险评估 |
| 股权管理 | 持续管理，及时了解被投资企业经营和财务状况 |
| 风险准备金 | 按有关规定提取 |

## 与破产实务的关联

### 1. 参与破产重整的主要方式

金融资产投资公司可以通过以下方式参与企业破产重整：

| 方式 | 说明 |
|------|------|
| 收购债权 | 收购银行对债务人的债权，成为债权人参与重整 |
| 债转股 | 将收购的债权转为股权，成为重整后企业的股东 |
| 直接投资 | 以重整投资人身份直接投资企业股权 |
| 债权处置 | 对未能转股的债权进行重组、转让和处置 |

### 2. 与破产重整中债转股的衔接

企业破产重整中，债转股是常见的债务清偿方式。金融资产投资公司作为专业机构：
- 可以收购银行债权后参与重整，将债权转为股权
- 可以与其他重整投资人合作，提供债转股专业服务
- 可以利用其银行背景，协调原债权银行支持重整

### 3. 不良资产处置

金融资产投资公司在破产程序中可以发挥不良资产处置的专业优势：
- 收购银行在破产企业中的不良债权
- 通过破产程序实现债权清收
- 对未能转股的债权进行市场化处置

### 4. 与《商业银行资本管理办法》的衔接

金融资产投资公司作为商业银行的主要股东发起设立，其自身也受到资本充足率监管。2022年银保监会发布的《金融资产投资公司资本管理办法（试行）》规定：
- 核心一级资本充足率不得低于5%
- 一级资本充足率不得低于6%
- 资本充足率不得低于8%
