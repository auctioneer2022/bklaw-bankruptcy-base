# 刑民交叉

本模块收录破产程序中涉及的刑民交叉问题相关资料，重点整理与破产相关的刑法条文、司法解释、规范性文件及典型案例。

## 收录范围

- [《刑法》第162条：妨害清算罪](xing-fa-di-162-tiao-fang-hai-qing-suan-zui.md) — 清算过程中隐匿财产、虚伪记载、违规分配
- [《刑法》第162条之二：虚假破产罪](xing-fa-di-162-tiao-zhi-er-xu-jia-po-chan-zui.md) — 通过隐匿财产、虚构债务实施虚假破产
- [《刑法》第162条之三：隐匿、故意销毁会计凭证、会计账簿、财务会计报告罪](xing-fa-di-162-tiao-zhi-san-yin-ni-hui-jin-hui-ji-ping-zheng-zui.md) — 隐匿或销毁依法应当保存的会计资料
- [《刑法》第175条之一：骗取贷款、票据承兑、金融票证罪（涉破产欺诈）](xing-fa-di-175-tiao-zhi-yi-pian-qu-dai-kuan-zui.md) — 以欺骗手段取得金融机构信用

## 内容说明

刑民交叉问题是破产实务中的难点和热点，涉及破产程序与刑事程序的协调、涉刑财产的处理等复杂问题。管理人发现犯罪线索时，应当及时向司法机关移送。
