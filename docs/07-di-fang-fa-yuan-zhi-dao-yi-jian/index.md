# 法院指导意见

本模块收录各省、自治区、直辖市高级人民法院及中级人民法院发布的与破产审判相关的司法文件、指导意见、解答等。这些文件是各地法院结合本地实际情况，对破产法律适用问题的具体细化和补充，对于指导当地破产审判实践具有重要参考价值。

## 内容说明

地方法院司法文件是对国家层面破产法律、司法解释的具体化和补充，具有以下特点：

1. **地域性**：针对本地区的实际情况和审判实践制定
2. **操作性**：对国家层面较为原则的规定进行细化，增强可操作性
3. **创新性**：在法律规定框架内，结合本地实践进行制度创新
4. **参考性**：对其他地区同类案件的处理具有参考借鉴价值

## 使用建议

- 在具体案件处理中，应优先适用案件受理法院所在地的地方法院司法文件
- 注意地方法院文件与国家层面法律、司法解释的协调一致
- 关注各地文件之间的差异，了解不同地区的审判思路和尺度
