## 正文

河南省高级人民法院：

你院《关于明确房地产领域矛盾纠纷案件裁判标准等问题的请示》（豫高法〔2022〕208号）收悉。经研究，现批复如下：

**一、** 建设工程价款优先受偿权、抵押权以及其他债权之间的权利顺位关系，按照《最高人民法院关于审理建设工程施工合同纠纷案件适用法律问题的解释（一）》第三十六条的规定处理。

**二、** 商品房消费者以居住为目的购买房屋并已支付全部价款，主张其房屋交付请求权优先于建设工程价款优先受偿权、抵押权以及其他债权的，人民法院应当予以支持。

只支付了部分价款的商品房消费者，在一审法庭辩论终结前已实际支付剩余价款的，可以适用前款规定。

**三、** 在房屋不能交付且无实际交付可能的情况下，商品房消费者主张价款返还请求权优先于建设工程价款优先受偿权、抵押权以及其他债权的，人民法院应当予以支持。

## 条文解读

### 第一条 建设工程价款优先受偿权等权利顺位

!!! note "条文解读"
    本条引用了《最高人民法院关于审理建设工程施工合同纠纷案件适用法律问题的解释（一）》第三十六条的规定，明确了建设工程价款优先受偿权、抵押权和其他债权之间的顺位关系：
    
    1. **建设工程价款优先受偿权** 优于 **抵押权** 和 **其他债权**
    2. **抵押权** 优于 **其他债权**
    
    但本批复第二条、第三条规定了商品房消费者的权利可以进一步优先于建设工程价款优先受偿权。

### 第二条 房屋交付请求权的优先保护

!!! note "条文解读"
    本条是本批复的核心条款，确立了商品房消费者的**房屋交付请求权**的最高优先地位。
    
    **适用条件**：
    
    | 要件 | 具体要求 |
    |------|----------|
    | 主体 | 商品房消费者（自然人，为生活消费需要） |
    | 目的 | 以居住为目的 |
    | 付款 | 已支付全部价款（或在一审法庭辩论终结前支付剩余价款） |
    | 主张 | 房屋交付请求权 |
    
    **优先对象**：
    
    - 建设工程价款优先受偿权
    - 抵押权
    - 其他债权
    
    **注意**：
    - "以居住为目的"排除了投资性购房
    - "已支付全部价款"包括一次性付款和已付清全部按揭贷款
    - 部分付款的消费者，在一审辩论终结前付清剩余价款即可适用

### 第三条 价款返还请求权的优先保护

!!! note "条文解读"
    本条规定了在房屋**无法交付**情况下的消费者权利保护。
    
    **适用条件**：
    
    | 要件 | 具体要求 |
    |------|----------|
    | 主体 | 商品房消费者（自然人，为生活消费需要） |
    | 前提 | 房屋不能交付且无实际交付可能 |
    | 主张 | 价款返还请求权 |
    
    **优先对象**：
    
    - 建设工程价款优先受偿权
    - 抵押权
    - 其他债权
    
    **注意**：
    - 本条适用于"烂尾楼"等房屋无法交付的情形
    - 消费者可以主张退还已付购房款
    - 退还的购房款请求权优先于所有其他权利

## 权利顺位体系

根据本批复及相关法律规定，商品房相关的权利顺位如下：

```
┌─────────────────────────────────────────────┐
│         商品房消费者权利（最高优先）           │
│  ┌─────────────────────────────────────────┐ │
│  │ 房屋交付请求权（房屋可交付时）            │ │
│  │ 价款返还请求权（房屋不能交付时）          │ │
│  └─────────────────────────────────────────┘ │
├─────────────────────────────────────────────┤
│      建设工程价款优先受偿权                   │
├─────────────────────────────────────────────┤
│            抵押权                            │
├─────────────────────────────────────────────┤
│          其他债权（普通债权）                  │
└─────────────────────────────────────────────┘
```

!!! warning "重要提示"
    商品房消费者的权利处于**最高优先地位**，甚至优先于建设工程价款优先受偿权和抵押权。这是对消费者居住权这一基本生存权的特殊保护。

## 与破产程序的关联

### 房地产企业破产中的适用

本批复在房地产企业破产案件中具有重要意义：

#### 1. 破产重整程序

- 消费者的房屋交付请求权不因破产程序而受影响
- 重整计划应当保障已购商品房消费者的权益
- 重整计划草案中应优先安排已售商品房的续建和交付

#### 2. 破产清算程序

- 消费者的价款返还请求权优先于所有其他债权
- 在破产财产分配中，消费者的购房款返还请求权应当优先清偿
- 消费者的优先权不受《企业破产法》第一百一十三条规定的清偿顺序限制

#### 3. 与《企业破产法》的衔接

| 问题 | 说明 |
|------|------|
| 消费者权利的法律依据 | 本批复 + 《企业破产法》 |
| 权利性质 | 属于法定优先权，非普通破产债权 |
| 清偿顺序 | 优先于破产费用和共益债务之外的所有债权 |
| 与职工债权的比较 | 消费者购房款返还请求权优先于职工债权 |

!!! note "实务要点"
    在房地产企业破产案件中，管理人应当：
    
    1. **全面排查已售商品房情况** — 梳理消费者购房信息
    2. **区分消费者类型** — 区分居住消费与投资购房
    3. **评估房屋交付可能性** — 判断是否适用第二条或第三条
    4. **制定专项处置方案** — 在重整/清算方案中优先安排消费者权益
