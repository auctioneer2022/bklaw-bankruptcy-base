# 最高人民法院关于实行社会保险的企业破产后各种社会保险统筹费用应缴纳至何时问题的批复

> **发布信息**
> - 发布机关：最高人民法院
> - 文号：法复〔1996〕17号
> - 发布日期：1996年11月22日
> - 施行日期：1996年11月22日
> - 文件性质：司法解释（批复）
> - 时效性：现行有效

---

## 正文

四川省高级人民法院：

你院川高法〔1995〕167号《关于实行社会保险的企业破产后，各种社会保险统筹费用应缴纳至何时的请示》已收悉。经研究，现答复如下：

**参加社会保险的企业破产的，欠缴的社会保险统筹费用应当缴纳至人民法院裁定宣告破产之日。**

---

## 条文解读

### 核心规则

!!! note "条文解读"
    本批复的核心规则非常明确：
    
    | 要素 | 内容 |
    |------|------|
    | 适用对象 | 参加社会保险的企业 |
    | 费用类型 | 各种社会保险统筹费用（养老保险、医疗保险、失业保险、工伤保险、生育保险等） |
    | 缴纳截止日 | **人民法院裁定宣告破产之日** |
    | 费用性质 | 欠缴的社会保险统筹费用属于破产债权 |

### 关键时间节点

```
企业正常经营期间 ──→ 破产申请受理 ──→ 破产宣告 ──→ 破产程序终结
       │                  │               │
       │                  │               └── 社保费用缴纳截止日
       │                  │
       │                  └── 破产程序开始（重整/清算）
       │
       └── 正常缴纳社保费用
```

!!! warning "注意"
    社保费用的缴纳截止日不是：
    - ❌ 破产申请受理之日
    - ❌ 破产程序终结之日
    - ❌ 企业实际停止经营之日
    
    而是：
    - ✅ **人民法院裁定宣告破产之日**

### 与新《企业破产法》的衔接

本批复发布于1996年，当时适用的是1986年《企业破产法（试行）》。2007年新《企业破产法》施行后：

| 对比项 | 旧法（试行） | 新《企业破产法》 |
|--------|-------------|------------------|
| 社保费用性质 | 第一顺序清偿（职工工资和劳动保险费用） | 区分处理：职工个人账户部分优先，其他社保费用为第二顺序 |
| 清偿顺位 | 统一作为第一顺序 | 职工个人账户部分：第一顺序；其他社保费用：第二顺序 |
| 截止日计算 | 破产宣告之日 | 破产宣告之日（本批复仍适用） |

!!! note "条文解读"
    根据新《企业破产法》第一百一十三条：
    
    > （一）破产人所欠职工的工资和医疗、伤残补助、抚恤费用，所欠的应当划入职工个人账户的基本养老保险、基本医疗保险费用，以及法律、行政法规规定应当支付给职工的补偿金；
    > 
    > （二）破产人欠缴的除前项规定以外的社会保险费用和破产人所欠税款；
    
    因此，社保费用的清偿需要区分：
    
    | 社保费用类型 | 清偿顺位 | 法律依据 |
    |-------------|----------|----------|
    | 应划入职工个人账户的基本养老保险、基本医疗保险 | 第一顺序（职工债权） | 《企业破产法》第113条第1款第1项 |
    | 其他社会保险统筹费用（单位缴纳部分） | 第二顺序（社保和税款） | 《企业破产法》第113条第1款第2项 |

## 实务要点

### 管理人的工作

管理人在处理社会保险统筹费用时应当：

1. **查明欠缴情况** — 向社保经办机构查询企业欠缴的社保费用明细
2. **区分费用类型** — 区分单位缴纳部分和个人缴纳部分
3. **计算截止日** — 以人民法院裁定宣告破产之日为截止日
4. **申报债权** — 将社保费用作为破产债权申报
5. **按顺位清偿** — 按照《企业破产法》规定的清偿顺序进行分配

### 社保经办机构的权利

| 权利 | 说明 |
|------|------|
| 债权申报权 | 社保经办机构作为债权人，有权申报欠缴的社保费用 |
| 优先受偿权 | 职工个人账户部分的社保费用享有优先受偿权 |
| 参与分配权 | 其他社保统筹费用按照第二顺序参与分配 |

### 实务中的争议问题

| 争议问题 | 实务观点 |
|----------|----------|
| 破产受理后至破产宣告期间的社保费用 | 根据本批复，应缴纳至破产宣告之日，因此该期间的费用也应缴纳 |
| 破产重整期间的社保费用 | 重整期间企业继续经营，应当正常缴纳社保费用 |
| 滞纳金的处理 | 参照法释〔2012〕9号批复，破产受理前的滞纳金属于普通债权，受理后的不属于破产债权 |

### 清偿顺序对照

```
破产财产
    │
    ├── 破产费用和共益债务（随时清偿）
    │
    ├── 第一顺序：职工债权
    │       ├── 职工工资、医疗、伤残补助、抚恤费用
    │       ├── 应划入职工个人账户的基本养老保险、基本医疗保险 ⬅️
    │       └── 经济补偿金
    │
    ├── 第二顺序：社保和税款
    │       ├── 其他社会保险统筹费用（单位缴纳部分） ⬅️
    │       └── 税款
    │
    └── 第三顺序：普通破产债权
            ├── 普通债权
            └── 破产受理前的滞纳金
```

### 实务提示

在房地产企业破产等涉及大量职工安置的案件中，社保费用的处理尤为重要：

1. **职工安置方案** — 应当包含社保费用的补缴计划
2. **政府协调** — 积极协调社保经办机构，争取政策支持
3. **职工权益保障** — 确保职工个人账户部分的社保费用优先受偿
4. **信用修复** — 重整成功后，及时办理社保关系的恢复和接续
