# 地方法规

本模块收录各省、自治区、直辖市人民代表大会及其常务委员会制定发布的与破产相关的地方性法规。

## 收录范围

### 省级地方性法规

- 各地企业破产条例
- 各地优化营商环境条例中涉及破产的条款
- 各地公司条例中涉及清算的条款

### 设区的市级地方性法规

- 市级优化营商环境规定
- 市级企业帮扶相关规定

## 内容说明

地方法规是地方国家权力机关依法制定的规范性文件，在不与上位法相抵触的前提下，可以结合本地实际情况对破产制度作出补充性规定。
