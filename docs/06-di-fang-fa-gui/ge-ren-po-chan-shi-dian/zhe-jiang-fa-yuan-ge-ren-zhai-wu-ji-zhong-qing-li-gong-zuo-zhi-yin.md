# 浙江法院个人债务集中清理（类个人破产）工作指引（试行）

> **发文机关**：浙江省高级人民法院
> **通过日期**：2020年12月2日（浙江省高级人民法院审判委员会第2820次会议通过）
> **时效性**：现行有效
> **条文数量**：共61条

## 文书要点

《浙江法院个人债务集中清理（类个人破产）工作指引（试行）》是**全国首个省级层面的类个人破产工作指引**，于2020年12月2日由浙江省高级人民法院审判委员会通过。该指引共61条，分为"基本原则""管辖""申请和受理""财产申报""管理人""财产调查、核实""债权申报""债权人会议""债务清理""程序终结""法律责任"十一部分。

该指引旨在建立一套有效的区分机制，通过债务人申报、接受债权人质询、人民法院依职权调查核实、管理人调查核实等措施，将"有履行能力而拒不履行生效法律文书确定义务"的债务人（"老赖"）与"诚实而不幸"的债务人予以区分，在对前者强制执行的基础上，给后者以相对宽松的制度出路。

### 核心制度创新

| 制度 | 要点 |
|------|------|
| 适用范围 | 浙江省户籍，在浙江省内居住并参加浙江省内社会保险或缴纳个人所得税连续满三年的自然人 |
| 基本原则 | 依法合规、鼓励探索、府院联动 |
| 管理人制度 | 引入管理人进行财产调查，可由列入名册的机构及执业律师、执业注册会计师或政府部门公职人员担任 |
| 双重表决规则 | 债权人会议审议事项采用双重表决机制 |
| 豁免财产 | 合理确定"生活必需品"范围，债权人会议审议结论作为法院决定的重要依据 |
| 免责考察期 | **5年**行为考察期 |
| 债务免除 | 引导债权人作出附条件减免债务的承诺 |
| 执行衔接 | 实现个人破产制度与强制执行制度的充分衔接 |

## 全文

### 一、基本原则

**1.** 依法合规，开展个人债务集中清理工作，应当坚持法治思维和法治方式，在现行法律框架内，依法合规开展工作，保障各方当事人的合法权益。

**2.** 鼓励探索，积极探索通过附条件的债务免除、诚信财产申报、合理确定"生活必需品"以实现破产制度中豁免财产的制度目的等途径，在个人债务集中清理工作中充分探索个人破产的制度因素。

**3.** 府院联动，积极推动政府相关部门在财产登记、公职管理人、专项资金、信用体系建设等方面优化个人破产的制度环境。

### 二、管辖

**4.** 符合以下条件的基层人民法院可以开展个人债务集中清理工作：

（一）自然人债务人住所地、经常居住地或主要财产所在地在该基层人民法院辖区内；

（二）该基层人民法院有以该自然人作为被执行人的强制执行案件。

**5.** 债务人向两个以上符合条件的人民法院申请的，由最先立案的人民法院开展个人债务集中清理工作。

### 三、申请和受理

**6.** 具有浙江省户籍，在浙江省内居住并参加浙江省内社会保险或缴纳个人所得税连续满三年的自然人不能清偿到期债务，资产不足以清偿全部债务或者明显缺乏清偿能力，可以依照本指引申请开展个人债务集中清理工作。

个体工商户可以参照本指引进行债务集中清理。

**7.** 债务人申请个人债务集中清理的，应当由本人提交下列材料，并现场签名：

（一）个人债务集中清理申请书；

（二）财产状况申报；

（三）债权人清册；

（四）债务方面的证据、收入和支出的证据；

（五）诚信承诺书；

（六）法院认为需要的其他材料。

**8.** 人民法院可以在诉讼服务中心引入管理人工作人员，就个人债务集中清理工作的受理条件、程序、法律后果等事项向债务人进行释明和引导。

**9.** 人民法院在个人债务集中清理案件审查受理阶段，可以召集已知债权人听证会，向债权人释明个人债务集中清理程序在引入管理人进行财产调查、债务人财产申报等方面的程序利益，引导债权人作出附条件的债务免除承诺。

人民法院可以将债权人附条件的债务免除承诺作为启动个人债务集中清理工作的条件之一。

**10.** 人民法院在收到符合条件的个人债务集中清理申请之日起三十日内裁定是否受理。

**11.** 人民法院可以自裁定受理个人债务集中清理申请之日起二十日内发布受理公告。公告可以载明下列事项：

（一）债务人姓名；

（二）人民法院裁定受理个人债务集中清理申请的时间；

（三）限制债务人行为的决定；

（四）管理人姓名或者名称及地址；

（五）债权人会议召开的时间、地点和方式；

（六）人民法院认为需要公告的其他事项。

**12.** 债务人申请个人债务集中清理属于主动纠正失信行为，人民法院裁定受理个人债务集中清理申请的，可以依照《最高人民法院关于公布失信被执行人名单信息的若干规定》决定提前删除失信信息。

**13.** 人民法院受理后，对于以进行个人债务集中清理的自然人作为被执行人的执行案件，受理个人债务集中清理申请的人民法院在浙江省范围内可以向执行案件的共同上级法院申请集中指定执行。

共同上级法院一般应当集中指定执行。

**14.** 自人民法院受理个人债务集中清理申请之日起至程序终结之日或者债务人行为考察期满之日止，债务人不得有以下消费行为：

（一）乘坐交通工具时，选择飞机商务舱、头等舱、列车软卧、轮船二等以上舱位、G字头高速动车组旅客列车二等及其他动车组一等以上座位；

（二）在三星级以上宾馆、酒店、夜总会、高尔夫球场等场所进行消费；

（三）购买不动产或者新建、扩建、高档装修房屋；

（四）租赁高档写字楼、宾馆、公寓等场所办公；

（五）购买非经营所必需车辆；

（六）旅游、度假；

（七）子女就读高收费私立学校；

（八）支付高额保费购买保险理财产品；

（九）其他非生活和工作必需的高消费行为。

**15.** 自人民法院受理个人债务集中清理申请之日起至程序终结之日或者债务人行为考察期满之日止，债务人应当履行下列义务：

（一）妥善保管其占有和管理的财产、文书资料，并根据管理人要求及时完整移交；

（二）根据人民法院、管理人的要求，如实报告本人及其配偶、共同生活近亲属名下的财产和收入状况，说明财产和收入变动情况；

（三）根据人民法院、管理人的要求，报告债权债务清册；

（四）配合人民法院、管理人的调查，配合管理人的财产清查、接管和分配；

（五）列席债权人会议并如实回答债权人的询问；

（六）及时向人民法院、管理人报告个人通讯方式、住址等信息的变动情况；

（七）未经人民法院许可，不得出境；

（八）未经人民法院许可，不得担任任何企业的法定代表人、股东、董事、监事和高级管理人员；

（九）借款一千元以上或者申请等额信用额度时，应当向出借人或者授信人声明本人正处于个人债务集中清理期间；

（十）人民法院认为应当履行的其他义务。

**16.** 人民法院受理个人债务集中清理申请后，已经开始而尚未终结的涉及债务人财产权利的民事诉讼或者仲裁，应当由管理人代为参加。法律另有规定的除外。

**17.** 自人民法院受理个人债务集中清理申请之日起至程序终结之日止，涉及债务人财产权利的民事诉讼，应当向受理个人债务集中清理申请的人民法院提起。

### 四、财产申报

**18.** 债务人应当如实申报本人及其配偶、共同生活近亲属名下的财产状况，包括但不限于：

（一）收入、银行存款、现金、支付宝和微信等第三方支付工具中的财产、理财产品、有价证券及住房公积金账户资金等现金类资产；

（二）土地使用权、房屋等不动产；

（三）交通运输工具、机器设备、产品、原材料、个人收藏品及非个人生活必需品等动产；

（四）债权、股权、投资权益、基金份额、信托受益权、知识产权等财产性权利；

（五）其他具有处置价值的财产。

债务人的财产已出租、已设立担保物权等权利负担，或者存在共有、权属争议等情形的，应当一并申报。债务人的动产由第三人占有及债务人的不动产、特定动产、其他财产权等登记在第三人名下的，应当一并申报。

**19.** 债务人应当如实申报自人民法院受理个人债务集中清理申请之日前三年内的财产变动情况：

（一）赠与、转让、出租财产；

（二）在财产上设立担保物权等权利负担；

（三）放弃债权或者延长债权清偿期限；

（四）一次性支出五万元以上大额资金；

（五）因离婚而分割共同财产；

（六）提前清偿未到期债务；

（七）其他重大财产变动情况。

**20.** 债务人应当如实申报债权债务清册，包括债权人姓名、债权数额、债权性质、债权到期日、债权担保情况等。

### 五、管理人

**21.** 个人债务集中清理工作中，如有必要指定管理人的，可以由债权人及债务人共同协商，在列入名册的机构及执业律师、执业注册会计师或政府部门的公职人员中确定。

**22.** 管理人应当履行下列职责：

（一）调查核实债务人及其所扶养人的基本情况；

（二）接管与债务人财产状况相关的财产清单、凭证及债权债务清册等资料；

（三）调查债务人财产状况和受理个人债务集中清理申请之日起前三年的财产变动情况，制作债务人财产报告；

（四）提出对债务人生活必需品（豁免财产）范围的意见，调查、接管债务人可供分配的财产；

（五）拟定债务人财产分配方案，经债权人会议通过后实施分配；

（六）代表债务人参加涉及债务人财产的诉讼、仲裁或者其他依法需要由债务人参加的活动；

（七）提议召开债权人会议；

（八）管理、监督、协助个人重整计划的执行；

（九）对债务人的免责申请出具书面报告；

（十）调查核实债务人是否有违反本指引相关规定的行为；

（十一）调查核实债务人是否有应当撤销免责的情形；

（十二）人民法院认为管理人应当履行的其他职责。

**23.** 管理人可以持指定管理人决定书向公安、民政、社会保障、税务、市场监管等部门和金融、征信机构依法调取债务人相关信息资料，有关部门和机构应当协助调查。必要时，管理人可以申请人民法院签发调查令。

**24.** 管理人应当勤勉尽责，忠实执行职务。

**25.** 管理人依照本指引履行个人债务集中清理案件管理职责，由人民法院依照有关规定确定其报酬。

管理人应当按照规定为破产财产不足以支付破产费用的案件提供破产事务公益服务。

### 六、财产调查、核实

**26.** 人民法院依职权对债务人财产状况进行调查核实。

**27.** 管理人应当对债务人财产状况进行全面调查核实，制作财产调查报告。

**28.** 债权人可以对债务人财产状况提出质询，债务人应当如实回答。

**29.** 人民法院、管理人发现债务人存在未如实申报财产等不诚信行为的，可以驳回其个人债务集中清理申请，或者终结个人债务集中清理程序，并依法采取相应的强制措施。

### 七、债权申报

**30.** 人民法院裁定受理个人债务集中清理申请后，对债务人持有债权的债权人，可以依照本指引规定的程序行使权利。

**31.** 人民法院裁定受理个人债务集中清理申请时，应当确定债权人申报债权的期限。债权申报期限自发布受理公告之日起计算，最短不得少于三十日，最长不得超过六十日。

**32.** 债权人应当在人民法院确定的债权申报期限内向管理人申报债权。债权人申报债权时，应当书面说明债权的数额和有无财产担保，并提交有关证据。

**33.** 在人民法院确定的债权申报期限内，债权人未申报债权的，可以在个人债务集中清理程序终结前补充申报；但是，此前已进行的分配，不再对其补充分配。为审查和确认补充申报债权的费用，由补充申报人承担。

**34.** 管理人收到债权申报材料后，应当登记造册，对申报的债权进行审查，并编制债权表。

**35.** 依照本指引规定无须申报的债权，管理人应当调查核实后列入债权表。

管理人应当将债权表提交第一次债权人会议核查。

债务人、债权人对债权表记载的债权有异议的，可以自收到债权表之日起十五日内向人民法院提起债权确认诉讼。

### 八、债权人会议

**36.** 依法申报债权并经管理人审查编入债权表的债权人有权参加第一次债权人会议，核查债权并行使表决权。

经人民法院裁定确认或者经确权诉讼判决确认的债权人为债权人会议成员，有权参加债权人会议并行使表决权。

**37.** 第一次债权人会议由人民法院召集，自债权申报期限届满之日起十五日内召开。

后续的债权人会议，在人民法院认为必要时，或者管理人、债权人委员会、所代表债权额占债权总额四分之一以上的债权人提议时召开。

**38.** 召开债权人会议，管理人应当提前十五日通知已知债权人。

**39.** 债权人会议可以对下列事项进行表决：

（一）核查债权；

（二）审议通过债务人生活必需品（豁免财产）清单；

（三）审议通过债务人财产分配方案；

（四）审议通过债务清理方案；

（五）选任和更换债权人委员会成员；

（六）人民法院认为需要由债权人会议表决的其他事项。

**40.** 债权人会议的决议，由出席会议的有表决权的债权人过半数通过，并且其所代表的债权额占对表决事项有表决权的债权总额的二分之一以上。但本指引另有规定的除外。

债权人认为债权人会议的决议违反法律规定，损害其合法权益的，可以自债权人会议作出决议之日起十五日内，请求人民法院裁定撤销该决议并责令债权人会议依法重新作出决议。

**41.** 债权人会议可以决定设立债权人委员会。债权人委员会由债权人会议选任的债权人代表组成。债权人委员会成员为单数且不得超过九人。

债权人委员会成员应当经人民法院书面确认。

**42.** 债权人委员会行使下列职权：

（一）监督债务人财产的管理和处分；

（二）监督债务人可供分配财产的分配；

（三）提议召开债权人会议；

（四）债权人会议委托的其他职权。

### 九、债务清理

**43.** 债务清理方案应当包括下列内容：

（一）债务人的基本情况；

（二）债务人财产状况；

（三）债务人生活必需品（豁免财产）清单；

（四）债权分类及债权调整方案；

（五）债务清偿方案；

（六）债务免除方案；

（七）信用修复方案；

（八）行为考察期及考察期内债务人的义务；

（九）债务清理方案的执行期限；

（十）有利于债务人重整的其他措施。

**44.** 债务清理方案中的债务清偿方案应当符合下列要求：

（一）债务清偿顺序符合法律规定；

（二）同类债权按比例清偿；

（三）不损害国家利益、社会公共利益或者他人合法权益。

**45.** 债务清理方案经债权人会议表决通过，并经人民法院裁定认可的，对债务人和全体债权人均有约束力。

**46.** 债务清理方案经债权人会议表决未获通过的，债务人或者管理人可以请求人民法院裁定批准。人民法院经审查认为债务清理方案符合本指引第四十四条规定的，应当裁定批准。

**47.** 债务清理方案未获得通过且未依照本指引第四十六条的规定获得人民法院批准的，或者已通过的债务清理方案未获得人民法院批准的，人民法院应当裁定终结个人债务集中清理程序。

**48.** 债务清理方案执行完毕，债务人申请免除剩余债务的，管理人应当对债务人是否存在不得免除的债务以及不得免除未清偿债务的情形进行调查，征询债权人和破产事务管理部门意见，并向人民法院出具书面报告。

人民法院根据债务人申请和管理人报告，裁定是否免除债务人未清偿债务。

**49.** 有债权人不同意免除债务人剩余债务或者将设置行为考察期作为同意免除剩余债务的条件的，行为考察期为裁定终结个人债务集中清理程序后的**五年**。

行为考察期内，债务人应当继续履行本指引第十五条规定的义务。

**50.** 债务人在行为考察期内违反本指引第十五条规定的，人民法院可以决定延长行为考察期，但延长期限不超过二年。

**51.** 行为考察期届满，债务人申请免除未清偿债务的，人民法院应当根据债务人申请和管理人报告，裁定是否免除债务人未清偿债务。

人民法院裁定免除债务人未清偿债务的，应当将裁定书送达债权人和债务人，并予以公告。

**52.** 债权人或者其他利害关系人发现债务人通过欺诈手段获得免除未清偿债务的，可以申请人民法院撤销免除未清偿债务的裁定。

**53.** 人民法院裁定不免除债务人未清偿债务的，或者撤销免除债务人未清偿债务裁定的，债权人可以向债务人继续追偿债务。

### 十、程序终结

**54.** 有下列情形之一的，人民法院应当裁定终结个人债务集中清理程序：

（一）债务人已清偿全部到期债务的；

（二）第三人为债务人清偿全部到期债务的；

（三）债务清理方案执行完毕的；

（四）债务清理方案未获得通过且未获得人民法院批准的；

（五）人民法院认为应当终结个人债务集中清理程序的其他情形。

**55.** 人民法院依照本指引第五十四条第一项规定裁定终结个人债务集中清理程序的，应当同时作出解除限制债务人行为的决定。

**56.** 人民法院裁定终结个人债务集中清理程序的，应当予以公告。

### 十一、法律责任

**57.** 债务人违反本指引规定，有下列行为之一的，由人民法院依法予以训诫、拘传、罚款、拘留；构成犯罪的，依法追究刑事责任：

（一）拒不配合调查，拒不回答询问，或者拒不提交相关资料的；

（二）提供虚假、变造资料，作虚假陈述或者误导性陈述的；

（三）故意隐匿、转移、毁损、不当处分财产或者财产权益的；

（四）其他妨害个人债务集中清理程序的行为。

**58.** 债务人违反本指引规定，存在未如实申报财产等不诚信行为的，人民法院可以裁定不免除债务人未清偿债务。

**59.** 债权人违反本指引规定，有下列情形之一的，由人民法院依法予以训诫、罚款、拘留；构成犯罪的，依法追究刑事责任：

（一）虚构债权或者虚报债权数额的；

（二）明知债务人处于个人债务集中清理程序仍然向其提供借款，导致其他债权人利益受损的；

（三）明知债务人虚假申报债务仍然予以确认的；

（四）其他妨害个人债务集中清理程序的行为。

**60.** 管理人未勤勉尽责、忠实执行职务，给债务人、债权人或者其他利害关系人造成损失的，应当承担赔偿责任。

**61.** 有关单位和个人违反本指引规定，拒不协助调查，或者拒不履行协助义务的，由人民法院依法予以训诫、罚款、拘留；构成犯罪的，依法追究刑事责任。

## 核心要点

### 适用条件

| 项目 | 内容 |
|------|------|
| 户籍要求 | 具有浙江省户籍 |
| 居住要求 | 在浙江省内居住 |
| 社保/个税要求 | 参加浙江省内社会保险或缴纳个人所得税连续满三年 |
| 破产原因 | 不能清偿到期债务，资产不足以清偿全部债务或明显缺乏清偿能力 |
| 管辖法院 | 债务人住所地、经常居住地或主要财产所在地的基层人民法院 |

### 核心制度

| 制度 | 要点 |
|------|------|
| 豁免财产 | "生活必需品"范围由债权人会议审议，结论作为法院决定依据 |
| 免责考察期 | **5年**行为考察期，可延长不超过2年 |
| 管理人 | 可由机构、执业律师、执业注册会计师或政府部门公职人员担任 |
| 表决规则 | 双重表决规则：过半数债权人+代表债权额二分之一以上 |
| 债务免除 | 引导债权人作出附条件减免债务承诺 |
| 执行衔接 | 与强制执行制度充分衔接，可集中指定执行 |

### 债务人行为限制

自受理之日起至程序终结或行为考察期满之日止，债务人不得：
- 乘坐飞机商务舱/头等舱、列车软卧、高铁一等座等
- 在三星级以上宾馆、夜总会、高尔夫球场消费
- 购买不动产、新建/扩建/高档装修房屋
- 租赁高档写字楼、宾馆、公寓办公
- 购买非经营必需车辆
- 旅游、度假
- 子女就读高收费私立学校
- 支付高额保费购买保险理财产品

## 与破产实务的关联

### 与深圳个人破产条例的比较

| 对比项 | 浙江工作指引 | 深圳个人破产条例 |
|--------|-------------|-----------------|
| 法律性质 | 法院工作指引（类个人破产） | 地方性法规（正式个人破产制度） |
| 适用地域 | 浙江省 | 深圳经济特区 |
| 社保要求 | 浙江省内社保或个税连续满3年 | 深圳社保连续满3年 |
| 免责考察期 | 5年 | 3年（可提前结束） |
| 豁免财产 | "生活必需品"，无明确金额上限 | 累计不超过20万元 |
| 程序启动 | 需有执行案件为前提 | 可直接申请 |
| 债务免除 | 需债权人同意（附条件承诺） | 法院裁定免除 |
| 管理人 | 公职管理人+市场化管理人 | 个人或机构均可 |

### 制度特色

- **执行程序衔接**：浙江模式是在现有执行程序框架内探索类个人破产制度，与强制执行制度充分衔接
- **区分机制**：建立"老赖"与"诚实而不幸"债务人的有效区分机制
- **府院联动**：积极推动政府相关部门在财产登记、公职管理人、专项资金、信用体系建设等方面提供支持
- **债权人主导**：强调债权人意思自治，债务免除需债权人作出附条件承诺
