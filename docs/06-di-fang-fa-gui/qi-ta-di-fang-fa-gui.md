# 其他地方性法规（府院联动机制）

## 概述

府院联动机制是指政府与法院之间建立的企业破产处置协调联动工作机制。根据国务院《优化营商环境条例》第三十三条规定，"县级以上地方人民政府应当根据需要建立企业破产工作协调机制，协调解决企业破产过程中涉及的有关问题"。

近年来，全国各地纷纷建立企业破产处置府院联动机制，统筹协调解决企业破产处置工作中的民生保障、社会稳定、财产接管、税收申报、资产处置、金融协调、信用修复、打击逃废债、变更注销、费用保障等问题，为实施市场化破产程序、防范化解重大风险、优化营商环境创造良好条件。

## 山东省潍坊市

### 关于建立企业破产处置府院联动机制的实施意见

> **发文机关**：潍坊市中级人民法院、潍坊市人民检察院、潍坊市发展和改革委员会等21个部门
> **文号**：潍中法〔2023〕2号
> **发布日期**：2023年
> **时效性**：现行有效

**核心内容**：

潍坊市建立了由21个部门参与的企业破产处置府院联动机制，成立潍坊市企业破产府院联动工作领导小组，组长由市政府分管副市长、市中级法院院长担任。

**主要任务**：

1. **破解难题，合力优化外部环境** — 以解决问题为导向，切实加强合作
2. **注重重整，努力盘活** — 充分认识破产重整制度对挽救企业和生产要素重组的重要作用
3. **维护稳定，促进法治** — 充分预判可能引发的金融风险和社会稳定问题
4. **创新机制，特事特办** — 有法律的依照法律办，没有法律的依照政策办，没有政策的依照惯例办，没有惯例的协商办理

**成员单位及职能分工**：

| 单位 | 主要职责 |
|------|----------|
| 市中级法院 | 召集府院联动机制协调会议；统筹研究解决企业破产处置中的问题；推动破产审判机制创新 |
| 市检察院、市公安局 | 依法加大破产"逃废债"犯罪行为打击力度；协同处置刑民交叉案件；维护稳定和社会管控 |
| 市审批服务局 | 为管理人依法办理破产事项所涉市场主体登记提供便利 |
| 市自然资源和规划局、市住建局 | 有效盘活土地资产；妥善认定资产权属；依法解除不动产保全措施 |
| 市人社局、市医保局 | 协调处理破产企业职工社会保险关系转移接续、欠缴社会保险补缴、职工工资拖欠、职工再就业等问题 |
| 市税务局 | 做好破产企业涉税辅导；依法申报欠缴税费及滞纳金债权；落实破产企业税务注销 |
| 市财政局、市国资委 | 探索构建破产案件援助基金制度；为"无产可破"案件提供资金支持 |
| 市地方金融监管局、人民银行、银保监分局 | 完善金融机构债权人内部决策流转程序；支持管理人依法接管破产企业账户；鼓励为重整企业提供信贷资金 |
| 市市场监管局 | 建立企业破产和退出状态公示制度；落实破产企业简易注销制度；依法及时变更重整企业股权 |

## 河北省石家庄市

### 关于建立企业破产处置府院联动机制进一步优化营商环境的通知

> **发文机关**：石家庄市人民政府办公室
> **文号**：〔2023〕-97
> **成文日期**：2023年8月28日
> **发布日期**：2023年9月1日
> **时效性**：现行有效

**核心内容**：

建立政府与法院常态化联动工作机制，形成"党委领导、政府支持、法院主导、部门联动"的一体化处置模式。

**工作机制**：

1. **会议机制** — 联席会议原则上每年召开一次；专题会议根据工作需要不定期召开
2. **信息共享机制** — 建立重大、敏感案件台账，定期通报相关信息
3. **重点案件专班机制** — 对市属国有企业破产处置和社会影响重大的疑难复杂破产案件，联合成立工作专班

**主要任务**：

- 积极引导困境企业寻求破产保护
- 完善企业破产工作风险预警机制
- 坚持精准识别、分类施策（重整/清算/执转破）
- 强化破产企业职工民生权益保障
- 依法解除破产企业财产查封、扣押、冻结措施
- 完善破产企业涉税事项处理机制
- 支持破产企业信用修复
- 加大破产"逃废债"打击力度
- 探索建立破产援助资金制度

## 山东省威海市

### 关于建立企业破产处置府院联动机制服务绿色低碳高质量发展的实施意见

> **发文机关**：威海市人民政府、威海市中级人民法院
> **文号**：威政发〔2023〕10号
> **成文日期**：2023年9月26日
> **发布日期**：2023年9月27日
> **时效性**：现行有效

**核心内容**：

建立健全市级层面企业破产处置府院联动机制，统筹协调解决企业破产处置工作中的民生保障、社会稳定、财产接管、税收处理、资产处置、金融协调、信用修复、打击"逃废债"、变更注销、费用保障等问题。

**工作任务**：

1. **保障破产企业职工合法权益** — 运用失业保险、最低生活保障等制度政策
2. **完善破产企业财产处置机制** — 有效盘活土地资产，依法解除财产保全措施
3. **加强金融和资金支持** — 鼓励为重整企业提供信贷资金，建立破产援助资金制度
4. **简化登记信息变更** — 落实破产企业简易注销制度，依法及时变更重整企业股权
5. **推行便利化涉税办理** — 保障发票供应，依法核销欠税，便利税务注销，支持纳税信用修复
6. **支持信用修复** — 协调开展重整、和解成功企业的信用修复工作
7. **加强招商引资** — 加大对重整企业招商引资的政策扶持力度
8. **依法打击"逃废债"犯罪** — 协同处置非法集资等刑民交叉案件
9. **加强管理人队伍建设** — 发挥破产管理人协会作用，提升履职能力
10. **防范化解重大风险** — 妥善处理国有企业及上市公司破产相关事宜

## 湖北省恩施州

### 关于建立常态化破产处置府院联动机制助力优化营商环境的意见

> **发文机关**：恩施州人民政府办公室
> **文号**：恩施州政办发〔2023〕18号
> **发布日期**：2023年7月27日
> **时效性**：现行有效

**核心内容**：

建立政府相关部门与法院"信息共享、合力处置"的常态化、制度化企业破产处置府院联动机制，以降低破产成本、压缩破产时间和尽最大努力挽救困境企业为核心。

**主要任务**：

1. **建立风险预警机制** — 对2年以上未能清偿到期债务的企业建立台账，对存在破产风险的企业进行预警
2. **健全分类处置机制** — 综合考虑企业资产状况、技术工艺、生产销售、市场前景等因素，按可以挽救、不能挽救分类施策
3. **构建联合处置机制** — 通过联席会议协调解决招商引资、职工安置、资产处置、税务处理、信用修复、企业注销、破产费用、信访维稳以及刑民交叉、打击逃废债等重点难点问题
4. **建立降低成本机制** — 确有困难的涉案企业申请免交诉讼费用的可以准许；降低鉴定、评估、审计等中介机构费用
5. **创新破产审判机制** — 探索集中管辖和集约化审理机制；探索公职管理人制度；建立管理人报酬与绩效挂钩的机制

## 与破产实务的关联

### 府院联动机制的核心功能

府院联动机制主要解决以下企业破产处置中的重点难点问题：

1. **职工安置** — 社会保险关系转移接续、欠缴社会保险补缴、职工工资拖欠、职工再就业
2. **涉税处理** — 税收债权申报、欠税核销、税务注销、纳税信用修复
3. **财产处置** — 土地资产盘活、资产权属认定、保全措施解除、产权瑕疵不动产过户
4. **金融支持** — 金融机构债权人决策、重整企业信贷支持、银行账户接管
5. **信用修复** — 重整/和解成功企业的信用修复、失信信息删除
6. **企业注销** — 简易注销、分支机构注销、股权质押/冻结情形下的注销
7. **费用保障** — 破产援助资金制度、"无产可破"案件费用支付
8. **打击逃废债** — 刑民交叉案件处置、非法集资案件协同
9. **招商引资** — 重整投资人招募、政府招商政策适用
