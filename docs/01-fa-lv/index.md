# 法律

本模块收录与企业破产相关的法律法规，包括《中华人民共和国企业破产法》及其配套法律。

## 收录范围

### 核心法律

| 文件名称 | 发布日期 | 主要内容 |
|----------|----------|----------|
| [《中华人民共和国企业破产法》](qi-ye-po-chan-fa.md) | 2006年8月27日 | 破产制度的基本法，规范企业破产程序 |

### 相关法律

| 文件名称 | 发布日期 | 主要内容 |
|----------|----------|----------|
| [《中华人民共和国公司法》](gong-si-fa.md) | 2023年12月29日（2023年修订） | 规范公司组织和行为，与破产程序密切相关 |
| [《中华人民共和国民事诉讼法》](min-shi-su-song-fa.md) | 2023年9月1日（2023年修正） | 破产案件审理程序未规定的部分适用民事诉讼法 |
| [《中华人民共和国民法典》](min-fa-dian.md) | 2020年5月28日 | 涉及债权债务、担保物权等基础民事法律关系 |
| [《其他法律中的破产相关条款》](qi-ta-fa-lv/index.md) | 多部法律 | 整理13部法律中与破产相关的核心条款，涵盖金融、劳动、税收、刑事等领域（共13份独立文件） |

### 配套法规

- 税务相关规定
- 工商登记规定
- 劳动社保规定
- 金融监管规定

## 使用说明

每部法律文件均包含：

1. **法律全文** — 完整的法律条文
2. **立法背景** — 法律的制定背景和目的
3. **条文解读** — 对重要条款的解释说明
4. **相关案例** — 与该法律相关的典型案例链接

## 法律体系关系

```
《企业破产法》（特别法）
    │
    ├── 《公司法》（一般法）— 公司组织和行为
    ├── 《民事诉讼法》（程序法）— 审理程序补充
    ├── 《民法典》（实体法）— 债权债务基础
    └── 行政法规/部门规章 — 配套实施规定
```
